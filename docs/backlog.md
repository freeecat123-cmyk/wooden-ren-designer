## 裝潢專屬計算器（v1.1+）

從 /about 拿掉的「即將推出」3 項——準備好再上：

### 🧱 木作隔間牆
- 輸入：長 × 寬 × 高、開門位置（無門 / 居中 / 偏移）
- 自動算：2×4 木骨架根數、矽酸鈣板片數（雙面）、釘子數量
- 工序：放樣 → 固定上下軌 → 立柱 → 封板 → 接縫處理
- 參考類比：天花板骨架（`/ceiling`）的結構演算法可重用

### 🪵 木地板估料
- 輸入：坪數、地板類型（海島型 / 超耐磨 / 實木）、是否邊角複雜
- 自動算：地板片數（含 5~10% 損耗）、收邊條長度、踢腳板長度、矽利康/膠用量
- 注意：地板有「方向性」損耗，邊角多會吃更多料

### 📏 線板 / 踢腳板
- 輸入：房間周長、線板款式（單線 / 雙線 / 寬版）、轉角數
- 自動算：線板支數（標準 8 尺一支）、外角數、內角數、釘距建議
- 加值：含含稅報價
