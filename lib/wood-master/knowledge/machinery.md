## 1. 桌上型／落地圓鋸機（Table Saw）

### 1.1 用途
- 縱切（Rip）：順木紋切長條
- 橫切（Cross-cut）：垂直木紋切短料
- 切槽、切榫肩、斜切（搭配輔具）
- 板材分割（夾板、木芯板裁大塊）

### 1.2 鋸片尺寸選擇
| 鋸片直徑 | 切深（垂直） | 適用場景 |
|---------|-------------|---------|
| 8 吋（210mm） | 約 55mm | 桌上型小機、攜帶現場 |
| 10 吋（254mm） | 約 80mm | 工作室標準（最普及） |
| 12 吋（305mm） | 約 100mm | 落地大機，可切厚硬木 |

- **10 吋是台灣木工最主流**，配件最齊、鋸片選擇最多
- 12 吋機台一般是 3HP+ 推力車型，切 75mm 以上實木較順

### 1.3 馬力對應
- **1HP（家用）**：220V 桌上型，可切 30mm 軟木為主，硬木會吃力
- **2HP（小型工作室）**：可推 40-50mm 實木縱切，多數中小機台
- **3HP+（職業級）**：5HP / 7.5HP 落地推台車型，切 75mm 以上硬木無壓力
- **三相 380V** 馬達比單相穩定、起動扭力大，工作室建議拉三相

### 1.4 鋸片齒數（最重要！買錯就毀料）
| 齒數 | 用途 | 切口品質 | 進料速度 |
|------|------|---------|---------|
| **24T**（FTG 平齒） | 縱切（Rip） | 粗 | 快 |
| **40T**（ATB 交錯齒） | **通用**（縱切+橫切） | 中 | 中 |
| **60-80T**（ATB） | 橫切（Cross-cut） | 細 | 慢 |
| **80-100T** | 細工、夾板貼皮、面材 | 極細（無爆邊） | 最慢 |

- **齒數越少 → gullet（齒間槽）越大 → 排屑越好**：24T 切厚實木不會燒料
- **齒數越多 → 每齒吃料越少 → 切面越光**，但 80T 切實木縱切會燒焦（排屑不夠）
- 一般家用一片 40T 萬用刀片就夠 80% 工作

### 1.5 刀片高度設定（kickback 預防關鍵）
- **教科書答案**：齒尖剛好露出板面 5-6mm（約 1/4 吋）
- **背景**：太低（只露 1/8 吋齒尖）反而 kickback 風險高（每齒切深變大、力距更難控制）
- **較高刀片**：每齒切削量小、向下分力大、料較不會被往後甩
- **新手安全準則**：刀齒整顆露出板面剛好（gullet 在板面上方），最常用的折衷

### 1.6 防 Kickback 五大裝置（缺一不可）
1. **分裂刀／撥料刀（Riving Knife / Splitter）**
   - 緊跟鋸片後方、跟鋸片同步升降
   - 防止木料切開後夾住鋸片後段、被往後彈飛
   - **2008 後 UL 規定新機必裝**，舊機沒有務必加裝
2. **防回踢爪（Anti-kickback Pawls）**
   - 分裂刀上的尖齒倒鉤
   - 木料若往後跑，爪子會抓住板面阻止飛出
3. **推板（Push Stick）**
   - 鋸片到圍欄距離 < 15cm 必用
   - 縱切寬條 < 30cm 也建議用
4. **防護罩（Blade Guard）**
   - 透明塑膠罩遮住鋸片上方
   - 切薄料時最該用，能擋住碎屑往臉部噴
5. **零間隙嵌板（Zero-Clearance Insert）**
   - 自製或購買，鋸片溝寬度剛好等於鋸片厚度
   - 防止薄料掉進嵌板縫隙、防止下方爆邊
   - 切實木貼皮、夾板貼皮必備

### 1.7 縱切寬條（Rip）操作 SOP
1. 板材一面整平、一邊整直角（先用 jointer），不能用毛邊靠圍欄
2. 圍欄校正平行於鋸片（前後誤差 < 0.1mm）
3. 刀片高度設定 5-6mm
4. **絕對不能徒手過鋸片附近**：切到剩 30cm 開始用推板
5. **窄條（< 50mm）**：搭配 Featherboard（羽毛壓板）壓住板面，推板貼著圍欄推
6. 站位：身體偏離鋸片中心線 30cm（避免 kickback 飛料路線）
7. **絕對不能** Miter gauge（直角規）+ 圍欄同時夾死料 → 切完瞬間夾住鋸片必甩飛

### 1.8 橫切（Cross-cut）操作 SOP
- **直角規（Miter gauge）**：適合短料（< 50cm），靠在鋸片左側
- **橫切板車（Crosscut sled）**：適合長料、大量重複切，自製滑車有兩條導軌
- **絕對不能用圍欄當橫切擋塊**：木料切完會夾在圍欄與鋸片之間 → 必 kickback
- 替代解：用 stop block 夾在直角規或 sled 前段，sled 與圍欄之間留 5mm 以上空隙

### 1.9 圓鋸機絕對禁止的操作
- 徒手切短料（< 30cm 一律用 sled 或 miter gauge）
- 直角規 + 圍欄同時夾住料
- 反向回拉切過的料（必 kickback）
- 拆掉分裂刀切槽（一定要切槽請用 dado 刀組 + 專用嵌板，且加 featherboard）
- 站在鋸片正後方延長線（kickback 飛料軌跡）
- 戴手套（手套會被鋸片捲入）
- 邊講話邊操作

### 1.10 知名品牌
- **SawStop**（美）：手指碰到自動煞車（5 毫秒內），業界最高安全標準，價格貴 2-3 倍但截肢風險近乎為零
- **Powermatic**（美）：橘色機身代表，職業級首選之一
- **Grizzly**（美）：CP 值高、台灣可代購
- **力山 REXON**（台）：BT2508RC 等桌上型代表，OEM 給歐美品牌
- **力勤**（台）：木工機械老牌，落地推台機
- **湧東木工機械**（台，1964 成立）：四面鉋、帶鋸、製材機等
- **仕興 OAV**（台）：SBW 系列帶鋸、推台鋸

## 2. 帶鋸機（Band Saw）

### 2.1 用途
- **曲線切割**：圓弧、心形、字模等不規則形狀
- **再切（Resaw）**：把厚板分薄成兩塊（如 50mm 厚分成兩片 22mm 板）
- **圓料縱切**：圓木切方
- **碎料安全切**：短料、不規則形狀比圓鋸機安全

### 2.2 喉深（Throat Depth）規格
喉深 = 鋸帶到立柱距離，決定能切多寬的料
| 喉深 | 適用場景 |
|------|---------|
| 14 吋（355mm） | 入門、小工作室標準（Delta 14、Rikon 10-326） |
| 17 吋（432mm） | 中型工作室，可 resaw 12 吋寬板 |
| 20 吋（508mm） | 職業級、可 resaw 16 吋以上寬板 |

### 2.3 鋸帶寬度（Blade Width）
鋸帶越寬 → 直線切越穩；鋸帶越窄 → 可切越小曲線
| 寬度 | 用途 | 最小可切曲線半徑 |
|------|------|----------------|
| 1/8 吋（3mm） | 細曲線、字模 | 5mm |
| 1/4 吋（6mm） | 一般曲線（最常用） | 15mm |
| 3/8 吋（10mm） | 大曲線+一般切割 | 35mm |
| 1/2 吋（12mm） | 縱切+小型再切 | 65mm |
| 3/4 吋（19mm） | 厚料再切（resaw）| 直線為主 |
| 1 吋（25mm） | 大型 resaw、製材 | 直線 |

### 2.4 TPI（Teeth Per Inch）選擇
原則：木料中至少要有 3-6 顆齒同時咬料
| TPI | 用途 |
|-----|------|
| 3 TPI | Resaw、厚料（>50mm）、軟木 |
| 4-6 TPI | 軟木一般切割 |
| 6-10 TPI | 硬木一般切割（橡木、楓木） |
| 10-14 TPI | 薄料、細曲線、密緻硬木（黑檀、紫檀） |
| 14+ TPI | 金屬切割（木工機請避開） |

**Skip tooth（跳齒）**：齒間距大、0° 前角，是 resaw 首選

### 2.5 設定三步驟（換鋸帶必做）
1. **張力（Tension）**
   - 一般範圍 15,000-30,000 PSI
   - 簡易測試：鋸帶在輪上，從側邊指彈，偏移約 3mm（1/8 吋）為合適
   - 太緊：鋸帶疲勞斷裂、輪軸磨損；太鬆：切不直、彎曲
2. **追蹤（Tracking）**
   - 鋸帶齒槽（gullet）最深點要對準上輪頂點（crown 中心）
   - 開機在低速下調整上輪傾角微調鈕
3. **輪導引（Guide Block / Bearing）**
   - **後推軸承（Thrust Bearing）**：與鋸帶背部留 1/32 吋（約 0.8mm）間隙
   - **側邊導引塊**：左右各與鋸帶留紙片厚（0.1mm）間隙
   - 上下兩組都要調

### 2.6 操作要點
- 上方鋸帶導引塊高度：壓在板面上方 5mm 內（越貼近板面越穩）
- 推料速度：慢勻速、不要硬推；硬推會讓鋸帶左右偏
- 曲線切：先順著切，遇到太彎時退出再從另一側切回來

### 2.7 安全警告
- **絕對不能反向退料**（切到一半往後拉），鋸帶會被勾出輪外
- 不要用手指按壓近鋸帶處（< 8cm 一律用推板）
- 切圓料前先在底部切平面或用 V-block 夾住，否則圓料會滾轉夾住鋸帶
- 鋸帶斷裂或脫輪時，**先關電源等鋸輪完全停**才開門

### 2.8 帶鋸機 vs 線鋸機（Scroll Saw）差別
| 項目 | 帶鋸機 | 線鋸機 |
|------|-------|-------|
| 鋸帶 | 環形連續鋸帶 | 直線往復鋸絲 |
| 切割厚度 | 50-300mm | < 50mm |
| 最小曲線 | 5mm | < 1mm（可切內部鏤空）|
| 力道 | 強，能切實木方料 | 弱，主要薄板 |
| 用途 | 大曲線、resaw | 細工藝品、拼圖、字模 |

## 3. 平刨機 / 手壓鉋（Jointer）

### 3.1 用途
- 把不平的板材一面整平（face joint）
- 把整完的這一面對應的相鄰一邊整直角（edge joint）
- **不能整厚！** 整厚是壓刨機（planer）的工作
- 一塊毛料的標準流程：jointer 整一面 → jointer 整一邊（直角） → planer 整厚 → table saw 切另一邊

### 3.2 規格選擇
| 機台寬度 | 適用場景 |
|---------|---------|
| 6 吋（152mm） | 入門、家用，可整 6 吋以下窄板 |
| 8 吋（203mm） | 工作室標準（多數實木板剛好） |
| 12 吋（305mm） | 職業級，可整寬板（拼板前不用先剖窄） |

- **8 吋是台灣木工的甜蜜點**
- 機台越長越好（infeed + outfeed table 越長 → 越能整長板）

### 3.3 進刀深度（Depth of Cut）
- **軟木（松、杉、檜）**：每次 1mm 內，極限 1.5mm
- **硬木（橡、楓、櫸、胡桃）**：每次 0.5mm，極限 1mm
- **寬板（face joint）**：再減半（0.25-0.5mm）
- **節疤密集 / 紋理交錯**：進刀降到 0.25mm，避免 tearout

### 3.4 排刀（Helical / Spiral Cutterhead）vs 直刀
| 項目 | 直刀（Straight Knife） | 排刀（Helical / Shelix） |
|------|---------------------|----------------------|
| 表面光滑度 | 一般，紋理交錯易 tearout | 極佳，幾乎無 tearout |
| 噪音 | 大（90+ dB） | 明顯小（多數情況不用戴耳塞） |
| 維護 | 兩三片刀整片換、需精準對刀 | 小型四方刀片，鈍了轉 90° 即可 |
| 換刀時間 | 30-60 分鐘 | 5 分鐘換單片，10 分鐘整圈轉 |
| 升級成本 | — | 加 NT$ 9,000-50,000 |
| 對交錯紋木材 | 必 tearout | 完美處理 |

- 預算許可、有處理花紋木需求 → 直接上 Shelix 排刀，不會後悔

### 3.5 操作姿勢
- 站位：偏左站，料從右往左推
- **左手在後段壓料**（outfeed table 上方），右手前段推進
- 推到一半時左手靠 outfeed 當基準、右手換成推塊（push pad）
- **永遠不能讓手指越過刀軸正上方**（即使料還在）
- 完成切後一手繼續壓料、另一手取另一支 push pad

### 3.6 紋理方向（Grain Direction）
- 觀察板側：紋理從左上往右下走時推料 → 順紋
- **逆紋**會 tearout，永遠檢查紋理再放料

### 3.7 安全警告
- **木料最短 25cm**（職業 jointer 規定 30cm，新手嚴格遵守 12 吋 = 30cm）
- **木料最薄 10mm**（太薄會被吹起、被刀打飛）
- 推板（push pad）必備，平刨永遠不徒手推到最後
- 永遠不刨弧形料（jointer 只整平面，弧形請用帶鋸或修邊機）
- **節疤密集處放慢、降進刀**（節疤崩裂會反彈）
- 不戴手套、長袖捲到手肘、不戴垂下的飾品
- 紅外線護指（Bladed Eye）若有務必開啟

### 3.8 絕對禁止
- 刨彎曲、扭曲嚴重的料（會被卡住反彈）
- 刨含釘、含金屬料（崩刀＋飛料）
- 同時兩塊料一起進
- 刨太短料（< 25cm 一律用其他工具或先拼長）

## 4. 壓刨機 / 厚薄機（Thickness Planer）

### 4.1 用途
- 把板材刨到指定厚度（兩面平行）
- 必須先有一面是平的（jointer 整過）才能放進壓刨
- **板材厚度均勻化**是壓刨唯一工作，不負責整平

### 4.2 規格選擇
| 機台寬度 | 適用場景 |
|---------|---------|
| 12 吋（305mm） | 入門桌上型（DeWalt DW735、Makita 2012NB）|
| 13 吋（330mm） | 進階桌上型 |
| 15 吋（381mm） | 落地中型，工作室主力 |
| 20 吋（508mm） | 職業級，可一次刨大寬板 |

### 4.3 每次刨削量（Depth of Cut）
- **硬木**：0.5mm 內（1/64 吋約 0.4mm 最安全）
- **軟木**：1mm 內
- **絕對上限**：1/8 吋（3.2mm），但實務上不要逼近
- **寬板**：再減半
- **接近最終厚度的最後 2-3 刀**：每次 0.2-0.3mm（避免最後 tearout 毀掉整片）

### 4.4 Snipe（前後段壓溝）解法
**現象**：板首和板尾各 5-10cm 出現比中段深 0.3-1mm 的下凹痕跡

**原因**：
- 板進機台時只有後滾輪壓住、前滾輪還沒咬 → 刀軸往下沉一點
- 板尾離開後滾輪、只剩前滾輪壓 → 同樣問題

**解法**：
1. **首尾各墊 3mm 廢料板（最有效）**：讓主板進出時始終有兩個滾輪壓住
2. **角度進料**：板與進料口呈 15-30° 斜角推進，滾輪先咬到角再轉正
3. **手扶法**：板進機時手往上扶板尾、板出時手往上扶板頭（讓滾輪持續壓住）
4. **延長台校正**：infeed/outfeed 延長板高度比主台低 0.2-0.5mm
5. **滾輪壓力調整**：太輕會 snipe，太緊會留壓痕，依機型微調
6. **慢進慢出、勻速**：不要急推

### 4.5 順紋 vs 逆紋判讀
- 入料前先看板側紋理：紋理由前向後往下走 = 順紋（OK）
- 紋理由前向後往上走 = 逆紋（會 tearout，調頭）
- **板兩面紋理可能相反**：哪面朝上看哪面，整完一面再翻

### 4.6 絕對不能刨
- **含釘 / 含金屬料**：崩刀、飛料、起火（先過金屬探測器）
- **再生料混雜物**（漆膜、強力膠殘留）：黏滾輪、降熱壞刀
- **太短料**：< 30cm 滾輪咬不住會被亂彈
- **太薄料**：< 6mm 會被吸進機台、捲入滾輪
- **彎曲嚴重料**（cup / bow > 5mm）：先用 jointer 整平一面再壓

### 4.7 操作要點
- 集塵管必接（壓刨產屑量極大，沒接 30 秒卡刀）
- 每次刨完一回，板翻面再刨對等量（兩面平衡刨）
- 多片同進：等寬料同時送，但每片要間隔 5cm（避免相互頂）

## 5. 角鑿機 / 方榫機（Mortiser）

### 5.1 用途
- 鑿方榫眼（mortise），木工榫卯接合的關鍵機台
- 比手鑿快、比修邊機 + 鳩尾治具精度高
- 主要用在桌椅腳、抽屜框、門框等結構件

### 5.2 規格
方鑿（Hollow Chisel）規格：6mm / 9mm / 12mm / 15mm / 18mm
- **9mm / 12mm 是最常用**（搭配 18-25mm 厚母材）
- 鑿子尺寸 = 榫眼寬度 = 公榫頭厚度

### 5.3 構造
- **外四方鑿（Hollow Chisel）**：四面開刃的方管，負責切方角
- **內螺旋鑽（Auger Bit）**：螺旋鑽頭穿過方鑿中心，負責掏料排屑
- 方鑿先壓進木頭切方角 → 螺旋鑽把中央木屑捲出 → 留下方形孔

### 5.4 鑿與鑽的間隙調整（最關鍵設定）
- **間隙**：螺旋鑽應比方鑿口先突出 1.5-4.5mm（1/16" - 3/16"）
- **太緊（< 0.5mm）**：鑽頭碰到方鑿內壁、燒鑽、起煙、鑽尖斷
- **太鬆（> 5mm）**：榫眼底部毛邊、鑽頭可能折斷
- **錢幣法**：用 1 元硬幣（約 1.5mm）當墊片，方鑿先卡死，鑽裝到鑽尖剛好露出鑿口；拿掉硬幣、方鑿往下推到底再鎖緊
- **精準墊片法**：0.2mm 塞尺墊在方鑿基座下、上鑽頭抵住方鑿底鎖緊、抽塞尺 → 自然有 0.2mm + 1.3mm（鑿厚）的間隙

### 5.5 進刀方法
1. **先鑿兩端定位**：榫眼長 50mm 的話，先在 0mm 和 50mm 兩端各鑽一刀完整深度
2. **再鑿中間連接**：在中間每次移動 < 鑿寬的 80% 連續鑽，避免單側受力鑿子彎掉
3. **不要連續滿鑿**：每鑽 1-2 刀升起讓鑽頭排屑

### 5.6 母材厚度與榫眼深度
- **木頭高度設定**：扣出母材厚度 / 3 = 榫眼深度（傳統盲榫比例）
- 例：30mm 厚的桌腳 → 榫眼深 20mm（厚度 2/3）→ 公榫頭長 20mm
- 厚度 2/3 是榫眼深（盲榫），不是 1/3

### 5.7 安全警告
- 鑿身會發燙：連續鑽 3-4 個榫眼後讓機台休息 30 秒
- 母材必須夾緊（用機台自帶夾具或 G 型夾），不能徒手扶
- 不要鑽近木邊（< 10mm 必裂）
- 鑽到底要留 2-3mm 不打穿（盲榫）

## 6. 砂磨機（Sander）

### 6.1 五大類型
| 類型 | 中文名 | 用途 | 易留刮痕？ |
|------|-------|------|----------|
| Belt Sander | 帶式（皮帶機） | 大面積、快速去料、整平 | 易（要謹慎） |
| Random Orbit | 軌道式（隨機軌跡） | **通用王者**，平面 + 修飾 | 幾乎無 |
| Sheet Sander | 振動式（方形紙） | 細修平面、邊角 | 中（直線紋）|
| Disc Sander | 盤式（落地） | 邊角整形、磨小圓弧 | 中 |
| Spindle Sander | 軸式（垂直軸） | 內凹曲線、孔內研磨 | 低 |

### 6.2 砂紙號數順序（金科玉律）
**80 → 120 → 180 → 240 →（修飾）320 / 400**

- **跳號禁忌**：80 直接跳 180 = 永遠磨不掉 80 的刮痕（120 顆粒才有辦法清掉 80 的痕）
- **黃金比例**：下一號不超過上一號的 50%（80→120 = +50% OK；80→180 不行）
- **不要超過 320**：> 320 會把木孔壓平、染色不吃色、塗料附著差

### 6.3 木材最終研磨號數對照
| 木種 | 收尾號數 | 備註 |
|------|---------|------|
| 松木 | 240 | 軟木怕磨毛、不要過度 |
| 橡木 | 180 | 木孔大、過細塗料更難滲入 |
| 楓木 | 220 | 緻密、可磨到 220 |
| 檜木 | 220 | 香味會被磨掉，不要過度 |
| 黑胡桃 | 220 | 油性木，220 後可上油 |
| 面漆前 | 320 | 上 PU、Lacquer 前最後一道 |
| 油性塗料前 | 220 | 油可滲入孔徑，太細反而吃不深 |

### 6.4 平面 / 曲面 / 邊角對應
- **大平面（桌面、櫃側板）**：軌道式（Random Orbit），5 吋盤最通用
- **超大平面（門板拼板）**：帶式（Belt）粗磨 → 軌道式收尾
- **小角落、邊角**：振動式（Sheet）方形紙好塞
- **內凹曲線（鏤空、桌椅腳曲線）**：軸式（Spindle）
- **外凸曲線**：軌道式 + 軟海綿背板
- **小工件邊緣整形**：盤式（Disc）

### 6.5 操作要點
- 力道：靠機台自重，**不要往下壓**（壓了會留軌跡刮痕）
- 速度：慢勻速移動，每 2-3cm/秒
- 紋理方向：軌道式雖然「隨機」，最後一刀仍要順紋
- **每次升號數前用刷子或吸塵器清乾淨上一號的灰**（沒清會混入留刮痕）
- 換號數時把上一號的痕跡完全磨掉再升

### 6.6 安全
- 必戴 N95 防塵口罩（細砂塵是矽肺主因）
- 接集塵
- 軌道式機尾不要靠近電線
- 帶式不要單點久磨（瞬間燒焦凹一塊）

## 7. 修邊機 / 雕刻機（Router / Trimmer）

### 7.1 用途
- 倒角（chamfer / round-over）
- 切槽（dado / groove）
- 修邊（flush trim）
- 鳩尾榫（dovetail）：搭配治具
- 刻字、3D 雕刻：搭配 CNC
- 鑲嵌（inlay）

### 7.2 軸柄（Shank）規格差別
| 軸柄 | 強度 | 適用 |
|------|------|------|
| 1/4 吋（6.35mm） | 1× | 小型修邊機、小銑刀、輕量倒角 |
| 1/2 吋（12.7mm） | 4×（截面積 4 倍） | 大型 plunge router、大銑刀、深切 |

- 1/2 吋軸柄不是 2 倍強，是 4 倍強（截面積平方關係）
- **1/2 吋振動明顯小** → 切面更光、銑刀壽命更長
- 大刀（直徑 > 25mm）一定要 1/2 吋軸柄，1/4 吋會折斷

### 7.3 轉速（RPM）
| 銑刀直徑 | 建議轉速 |
|---------|---------|
| < 25mm（1 吋） | 22,000-24,000 RPM（多數修邊機滿速） |
| 25-50mm（1-2 吋） | 18,000-22,000 RPM |
| 50-65mm | 12,000-18,000 RPM |
| > 65mm（2.5 吋以上） | 8,000-12,000 RPM（需要可調速大型 router） |

- 太快：邊緣燒焦、銑刀短命
- 太慢：切不乾淨、tearout
- **大刀必須降速**（線速度太高會解體飛出）

### 7.4 銑刀（Bit）類型
- **直刀（Straight Bit）**：切槽、開榫眼
- **半圓刀（Round-over）**：把直角邊修成 R 角
- **燕尾刀（Dovetail）**：鳩尾榫專用，14° 或 8°
- **修邊刀（Flush Trim）**：頂端帶軸承，沿模板修齊
- **羅馬線（Roman Ogee）**：S 型線條，櫃門面板裝飾
- **倒角刀（Chamfer）**：45° 斜邊
- **V 字刀（V-bit）**：刻字、CNC 雕刻
- **嵌合刀（Rabbeting）**：切階梯邊（搭門邊用）

### 7.5 進料方向（很重要！搞錯就 kickback）
- **手持修邊機**：銑刀往下看順時針轉 → 板邊在右側時 → 從**右往左**推進
- **修邊機檯（Router Table）**：銑刀往上看逆時針轉 → 從**右往左**推料
- **記憶法**：永遠 against 銑刀旋轉方向推進
- **Climb cut（順切，與旋轉同向）**：會被銑刀吸住往前甩，**修邊機檯禁用**（除非 featherboard 雙重壓料 + 極淺切深 + 大塊料）

### 7.6 安全要點
- 固定銑刀：1/4 吋軸柄要塞進夾頭至少 19mm（3/4 吋）深，不能只塞一半
- **薄料（< 12mm）必先用雙面膠或夾具固定**
- 修邊靠模板（template）：銑刀軸承沿模板邊滾動 → 形狀完全複製
- 多次淺切勝一次深切：每次切深 ≤ 銑刀直徑（如 6mm 銑刀每次 6mm 內）
- 銑刀換完先空轉 30 秒檢查震動
- 手不要放在銑刀前進路徑上（順時針 = 銑刀會把料往左推）

### 7.7 修邊機 vs 大型雕刻機差別
| 項目 | 修邊機（Trimmer） | 大型雕刻機（Plunge Router） |
|------|-----------------|---------------------------|
| 重量 | 1-1.5kg 單手 | 3-5kg 雙手 |
| 馬力 | 0.5-1 HP | 2-3.5 HP |
| 軸柄 | 1/4 吋為主 | 1/2 吋為主，可換 1/4 吋夾頭 |
| 用途 | 倒角、修邊、輕切槽 | 深切、大刀、模板複製、鳩尾治具 |
| 升降 | 固定底座 | Plunge（垂直下壓）+ 固定底座 |

### 8.1 木工 CNC（Computer Numerical Control）
| 機型 | 工作面積 | 主軸 | 適用 |
|------|---------|------|------|
| 6090（家用 CNC） | 600×900mm | 1.5-2.2kW 水冷 | 個人工作室、小招牌 |
| 1212 / 1325 | 1.2×1.2 / 1.3×2.5m | 3-5.5kW | 商用、家具廠 |
| Shopbot（美） | 多種 | 真空吸盤桌面 | 專業 |
| X-Carve（美） | 750×750mm | 桌上型 | DIY 入門 |
| Avid CNC | 客製 | 模組化 | 高 CP 半專業 |

- **真空吸盤桌**比夾具方便快 10 倍，但要配真空泵
- **ER20 夾頭**最常用，支援 1/4 吋與 1/2 吋
- 軟體：Vectric Aspire / VCarve、Fusion 360、Carbide Create

### 8.2 雷射雕刻（CO2 Laser）
| 功率 | 用途 | 木材切割上限 |
|------|------|------------|
| 40W（K40 中國機） | 雕刻為主、薄夾板（< 3mm） | 3mm |
| 60W | 雕刻為主、可切 5mm 軟木 | 5mm |
| 80W | 雕刻+切割 | 約 12-18mm 軟木一刀 |
| 100W | 雕刻+切割 | 15-20mm 軟木一刀 |
| 130-180W | 商用切割 | 25mm 以上 |

- **木材最大可切**：硬木 < 軟木 < 夾板

- 雷射禁切：PVC、ABS（含氯氣會殺死雷射管）、含金屬塗層板

### 8.3 數位繪圖機（Plotter / Cutter）
- 用於切貼紙、圖紙模板
- Cricut / Silhouette 是入門 DIY 級
- 木工常用：列印 1:1 圖紙貼在板材上當切割參考線

## 9. 鑽床（Drill Press）

### 9.1 桌上型 vs 落地型
| 項目 | 桌上型（Bench） | 落地型（Floor） |
|------|---------------|---------------|
| 高度 | 1m 以下，放桌上 | 1.6-1.8m 獨立 |
| 行程（quill travel） | 70-100mm | 120-160mm |
| 馬力 | 0.5-0.75 HP | 1-2 HP |
| 喉深 | 6-10 吋 | 12-16 吋 |
| 用途 | DIY、輕量 | 工作室、深孔、大孔 |

### 9.2 速度檔位（轉速）對照
木工常用速度（**重點**：鑽頭越大、木頭越硬，速度越慢）

| 鑽頭類型與直徑 | 軟木 RPM | 硬木 RPM |
|--------------|---------|---------|
| 麻花鑽 < 6mm（1/4 吋） | 3000 | 1500 |
| 麻花鑽 6-12mm | 2000 | 1000 |
| 麻花鑽 12-25mm | 1000-1500 | 500-750 |
| **平翼鑽（Forstner）< 25mm** | 1000-1500 | 500-1000 |
| **Forstner 25-50mm** | 500-1000 | 300-500 |
| **Forstner > 50mm** | 250-500 | 250 |
| 螺旋鑽（Auger，深孔） | 1500 | 750 |
| 階梯鑽（Step） | 750-1500 | 500-1000 |

### 9.3 平翼鑽（Forstner）vs 一般鑽差別
| 項目 | Forstner | 麻花鑽（Twist） | 螺旋鑽（Auger） |
|------|---------|---------------|---------------|
| 鑽孔底部 | 平底（極平整） | V 字尖 | V 字尖 |
| 排屑 | 慢，靠抬升 | 快，自動排 | 最快，螺旋拉屑 |
| 適合厚度 | 任何厚度（但深孔慢） | 任何 | 深孔（> 30mm） |
| 鑽淺孔（< 10mm）| ✓ 完美 | × 鑽尖會穿透 | × 不行 |
| 緊鄰邊孔 | ✓ 不爆邊 | × 易裂 | × |
| 速度 | 慢（必降速） | 快 | 中 |

- **Forstner 是木工神器**：鑽鉸鏈孔、安裝插銷孔、瓶蓋槽、平底盲孔
- 直徑 35mm 是廚櫃絞鏈杯標準

### 9.4 操作要點
- 木料夾緊：用 G 型夾或鑽床夾鉗，**永遠不能徒手扶**（特別是大鑽 / Forstner，會把整塊料轉飛起來）
- 鑽深孔分段：每鑽進 鑽頭直徑 × 1 深度後抬起讓排屑
- 鑽穿孔背面墊廢料：避免出口爆裂
- 校正主軸與工作檯垂直（90°），用直角規檢查

### 9.5 安全
- 不戴手套（會被夾頭捲入）
- 長髮綁起、不戴項鍊袖口
- Forstner 大鑽必須夾料並降速，違反者一定甩飛
- 主軸啟動前確認鑽頭鎖緊（用夾頭鑰匙）

## 10. 集塵與安全（Dust Collection & Safety）

### 10.1 集塵機規格（CFM = Cubic Feet per Minute）
| 機台 | 建議 CFM |
|------|---------|
| 修邊機、線鋸機 | 200-350 CFM |
| 砂磨機（軌道式） | 250-400 CFM |
| 圓鋸機 | **400-650 CFM**（細粉多） |
| 平刨機 / 壓刨機 | 600-800 CFM（產屑量大） |
| CNC | 600-1000 CFM |
| **理想值（最細粉收乾淨）** | 800-1000+ CFM |

- 桌上型 shop vac（吸塵桶）：80-150 CFM，搭配 2.5 吋管適合**單機台手持工具**
- 中型集塵機（1-2HP）：500-700 CFM，4 吋管
- 大型集塵機（3-5HP）：1500+ CFM，6 吋管，可一拖多

### 10.2 集塵管直徑
| 機台類型 | 管徑 |
|---------|------|
| 大機台（圓鋸、刨床、壓刨） | 4 吋（100mm） |
| 手持工具（修邊機、軌道砂磨機） | 2.5 吋（63mm） |
| CNC、雷射 | 4 吋 |
| 主幹道（一拖多）| 6 吋（150mm）|

- **誤用 2.5 吋接刨床**：CFM 從 233 跌到 91（-60%），等於沒接
- **誤用 4 吋接小修邊機**：流速太慢、細粉沉澱在管內

### 10.3 過濾等級
- **HEPA**（H13/H14）：過濾 0.3μm 微粒 99.97%，矽肺、過敏者首選
- **MERV 16**：商用級
- 沒過濾：細粉飛回工作室，戴口罩也吸到

### 10.4 機台連動
- **Long-Ranger 遙控**：開機台時集塵機自動啟動，3-5 秒延遲關閉（避免管內殘屑）
- iVAC、ClearVue、台灣 Festool / Bosch 也有自家系統
- 工作室建議標配，省電 + 省心

### 10.5 PPE（個人防護裝備）
| 裝備 | 規格 | 場景 |
|------|------|------|
| **安全眼鏡** | ANSI Z87+（高速衝擊認證） | 所有開機操作 |
| **耳罩 / 耳塞** | NRR 25dB+（圓鋸 105dB→80dB 安全範圍） | 圓鋸、刨床、修邊機 |
| **耳塞 + 耳罩雙重** | 合計 NRR 38 | CNC 長時間、雷射 |
| **N95 防塵口罩** | NIOSH N95 認證 | 砂磨、切 MDF、抽屜製作 |
| **PAPR 動力濾風頭盔** | 全臉防護 + 過濾 | 大量切硬木、車床、長時間 |
| **防護手套** | **不戴！** | 機台操作（會被捲入） |
| **皮圍裙** | 防火星 | 車床、雷射 |
| **包覆鞋** | 鋼頭最佳 | 全工作室 |

- **重點**：操作旋轉刀軸的機台（圓鋸、刨床、車床、修邊機、鑽床、CNC）**禁戴手套**
- 搬運生料、上漆、組裝時可戴手套

### 10.6 工作室安全規範
- 急救箱位置全員知道
- 滅火器（ABC 乾粉式）每 6 個月檢查
- **集塵機桶也會自燃**（細粉 + 火花 = 粉塵爆炸），桶內火花探測器有預算就裝
- 緊急停機鈕（紅蘑菇）位置明確
- 學徒 / 新人 30 天內每台機只能在師傅監督下操作

- **機械專拍**：北部桃園、中部太平、南部仁武有專門場
- **看貨重點**：
  - 通電試車聽馬達（軸承壞會嘎嘎叫）
  - 鋸軸 / 主軸跳動 < 0.1mm
  - 機身鑄鐵有沒有裂痕
  - 配件齊全（圍欄、嵌板、推板、鋸帶）
  - 三相 vs 單相要先確認自家電源

### 11.3 蝦皮便宜中國機台的雷區
**不建議買的雷品**：
- **無品牌「桌上型」圓鋸機**（NT$ 3,000-5,000）：
  - 馬達實際 0.3HP（標 1HP），切 30mm 軟木會卡停
  - 鋸軸跳動大，無法切直線
  - 圍欄鬆動、無法平行對位
  - 沒分裂刀，kickback 風險極高
- **山寨「電木銑」/ 修邊機**（NT$ 800-1500）：
  - 夾頭只支援 6mm 軸柄，銑刀選擇受限
  - 軸承品質差，3 個月開始漏粉
- **無認證雷射雕刻機 K40**（NT$ 8,000-12,000）：
  - 沒有水冷、雷射管壽命 2-3 個月
  - 沒排煙、有毒煙進工作室

**可以考慮的中國機台**：
- 6090 CNC（NT$ 50,000+）有正規供應商
- 大型 CO2 雷射 80W+（含水冷、含排煙）
- 買前先在 LumberJocks、台灣社團查網友開箱

### 11.4 採購建議流程
1. **先決定預算與工作室空間**（電力供應：單相 220V or 三相 380V）
2. **看二手出清**：很多工作室收掉，6-7 折好物
3. **大機台優先買中古職業級**（Powermatic、Delta、SCM 二手）勝過全新陽春機
4. **小機台、手持買新**（修邊機、軌道砂磨機）
5. **耗材通路熟練**（鋸片、銑刀、砂紙）：找一兩家固定供應商談折扣

## 結語

木工機械是**力 + 速度 + 慣性**的組合，每一次按下開關都帶著 1-3HP 馬達 + 數千 RPM 的能量。所有規格、安全裝置、操作 SOP 不是廠商隨便寫的，是過去百年血淚換來的教訓。

**三條鐵律**：
1. **不熟悉的機台，先看完手冊+影片+找師傅在旁，第一次別獨自操作**
2. **任何「我這樣切過很多次都沒事」的瞬間，下一次可能就是斷指的那次**
3. **PPE、推板、防護罩拿掉一次就會拿掉第二次** → 永遠不要拿掉

工具是延伸，不是賭注。做木工是長期事業，先安全才有作品。
