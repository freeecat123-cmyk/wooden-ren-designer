# 家具修補診斷 SOP（Repair & Restoration）

> 木頭仁 AI 木工大師客服知識檔 — 修補篇  
> 適用範圍：家具行 / 工作室等級的修復，不到博物館古董修復等級。  
> 核心理念：**先診斷，再下手**。台灣家具退貨第一名場景就是「搬家後開裂」「冷氣房翹曲」「客戶說榫鬆掉」，本檔把零散的含水率/伸縮空間資訊整合成可執行 SOP。

---

## 0. 診斷分類（先問這 4 組問題定位）

接到客戶投訴或自己家具壞了，**不要第一秒就開始修**。先問清楚以下 4 組問題，才知道要走哪一條 SOP，否則白花力氣甚至越修越糟。

### 0.1 幾何問題（Where）
- **裂在哪？**
  - 板面中央裂（多半是含水率變動）
  - 端頭裂 / 端裂（end check，乾燥太快、未封端）
  - 沿節疤裂（應力集中）
  - 順著膠合線裂（拼板膠合失敗）
- **翹在哪？**
  - 橫向翹（cup 瓦狀翹）
  - 縱向翹（bow 弓狀翹）
  - 對角翹（twist / wind 扭轉）
- **鬆在哪？**
  - 榫頭鬆（mortise & tenon loose）
  - 圓榫 / 木釘鬆（dowel loose）
  - 燕尾鬆（dovetail loose，少見但有）
- **斷在哪？**
  - 順紋裂斷（沿木紋方向，多半可救）
  - 橫紋斷（cross-grain failure，幾乎要換料）
  - 榫頭斷（最常見，可補可換）

### 0.2 環境問題（Where it lives now）
- **搬到哪個環境？** 從工作室搬到客戶家、客戶搬家、從一樓搬到頂樓、剛開始開冷氣房。
- **室內溫濕度？** 台灣理想 20-26°C / RH 50-65%。RH 低於 40% 木頭會縮、高於 75% 會膨。
- **靠近熱源？** 冷氣出風口正下方、暖爐旁、落地窗西曬位置 = 高風險。
- **地板材質？** 直接放磁磚冷地板 vs 木地板，含水率交換速率差 3 倍。

### 0.3 時間軸（When）
- **什麼時候做的？** 剛交貨 1 個月內出問題 = 含水率沒乾透 / 工法有疑慮。
- **什麼時候出問題？** 季節交替（5 月梅雨、9 月秋燥、12 月冷氣季）= 環境問題，不是你的鍋但要幫客戶解決。
- **多久發展成現在這樣？** 突然變化（一夜之間裂） vs 漸進變化（半年慢慢翹）。

### 0.4 結構（What it's made of）
- **實木 / 拼板 / 集成材 / 板材？**
  - 實木：寬度愈大愈會動，要看伸縮預留有沒有做。
  - 拼板：要看有沒有年輪交替，膠線是 Type II PVA 還是 hide glue？
  - 集成材 / 木芯板 / 夾板：基本上不會自己動，動了通常是受潮或膠失效。
- **榫接方式？** 通榫 / 半榫 / Domino / 木釘 / 螺絲？修補方式完全不同。
- **塗裝？** 油性 PU、水性 PU、油蠟、Shellac、刷漆？不同塗裝對水痕、白霧、修補染色容忍度天差地遠。

## 1. 常見問題 1：板面開裂（Split / Check）

### 1.1 症狀辨識
- **微裂（hairline crack，< 1mm）**：肉眼幾乎看不到，指甲滑過會卡。
- **中裂（1-5mm）**：看得到縫，光線會穿透。
- **大裂（> 5mm）**：手指能伸進去、結構強度受影響。
- **端裂（end check）**：木材端頭沿木紋方向短裂縫。
- **貫穿裂（through split）**：從一面裂到另一面、整塊板分兩半的趨勢。

### 1.2 成因
| 成因 | 機制 |
|---|---|
| 含水率劇變 | 出廠 12% → 客戶冷氣房降到 6%，木材橫向縮 1-2%，超過彈性極限就裂 |
| 伸縮空間沒預留 | 桌面四邊鎖死、面板釘死框架，木材想動但動不了 → 應力釋放成裂縫 |
| 徑切弦切混合 | 弦切（flat-sawn）橫向變形量是徑切（quarter-sawn）的 2 倍，混拼必裂 |
| 節疤應力集中 | 節疤周圍木紋扭曲，是天然弱點，乾縮時最先撕開 |
| 端頭未封蠟 | 端頭水分蒸發比側面快 10-12 倍，乾太快就 end check |

### 1.3 修補 SOP（依裂縫大小選方案）

#### 方案 A：微裂（< 1mm）→ CA 膠（瞬間膠）滲入
1. 用吸塵器或壓縮空氣清乾淨裂縫內粉塵。
2. 把表面磨砂粉（同種木材）撒進裂縫填底。
3. **滴入薄 CA 膠（thin CA / Starbond 細口注射器）**，膠會被毛細力吸入。
4. 等 30 秒，再滴一次中黏度 CA 膠把表面填平。
5. **CA 加速劑（accelerator）噴一下**，10 秒硬化。
6. 砂磨 320# → 400# → 600# 拋光，重塗保護層。

#### 方案 B：中裂（1-5mm）→ 環氧樹脂 + 木屑
1. 用刷子或壓縮空氣清乾淨裂縫，**裂縫不能潮濕**。
2. 用同種木材磨細木屑（細到像麵粉）。
3. **環氧樹脂（epoxy，台灣常用紅蝙蝠 / West System）依比例調好**。
4. 木屑混入環氧樹脂，調到「花生醬」濃稠度。
5. 用注射器（針頭剪掉一段）灌入裂縫，由低處灌起避免空氣囊。
6. 用塑膠刮刀壓緊、表面略凸（環氧樹脂收縮極小但還是會凹）。
7. **24 小時硬化**（紅蝙蝠 30 分快速型可 1 小時，但耐久度差一點）。
8. 砂磨 220# → 320# → 400# 齊平。
9. 補染色（如果顏色不對）→ 重塗。

#### 方案 C：大裂（> 5mm 或貫穿裂）→ 蝴蝶榫（Butterfly Key / Dutchman）

> **這是修補大裂的招牌做法，必學。** 蝴蝶榫不只填縫，更是「機械鎖死」防止裂縫繼續擴大。

##### 蝴蝶榫 Step-by-Step

**1. 設計蝴蝶形**
- **角度比例 1:6**（和燕尾榫一樣，避免太尖會崩、太胖咬合不夠）。
- **長度方向 = 跨越裂縫的方向**，要垂直於裂縫。
- **大小規範**：
  - 寬度 = 板厚 1-1.5 倍
  - 長度 = 寬度 1.5-2 倍
  - 厚度 = 板厚 1/3（不超過 1/3，否則弱化板材）

**2. 在板上畫位置**
- 跨裂縫對稱放置，蝴蝶腰部正壓在裂縫上。
- **間距 100-150mm 一片**（裂縫長就要多片）。
- **大裂兩端要各放一片**收尾，否則裂縫會繞過繼續走。

**3. 切蝴蝶榫片**
- 用對比色木材（黑檀 vs 楓木 / 胡桃 vs 橡木）製造視覺亮點，這是 George Nakashima 的經典手法。
- **切「先做榫片、再畫板上」** 比反過來精準很多。
- 帶鋸或修邊機切外形，順木紋（長纖維方向 = 蝴蝶長軸方向，這樣才有抗拉強度）。

**4. 挖榫眼**
- 板上描蝴蝶榫片輪廓（用刀刻，不要用鉛筆，鉛筆線會被砂磨吃掉）。
- 用修邊機 + 直刀銑床（straight bit）銑出 **深度 = 板厚 1/3** 的初坑。
- 修邊機只能銑出大致形狀，**轉角要鑿刀手工修**（修邊機刀直徑會留圓角）。
- 試組裝，蝴蝶片應該滑入 2/3 深度後卡住，這就是緊度剛好。

**5. 膠合**
- 環氧樹脂塗榫眼底面 + 側壁。
- 蝴蝶片敲入（橡膠槌或木槌墊布）。
- **不要敲到完全平**，留 1-2mm 凸出來。
- 24 小時硬化。

**6. 修齊**
- 手刨 / 砂磨齊平。
- 整面砂磨 220# → 320# → 400#。
- 重塗保護層。

##### 蝴蝶榫常見錯誤
- 角度太尖（< 1:8）→ 尖端崩。
- 角度太胖（> 1:4）→ 抓不住。
- 木紋方向錯（橫紋）→ 蝴蝶榫自己先裂。
- 深度太深（超過板厚 1/2）→ 板背面凸起或變形。
- 只放一片在中間 → 裂縫繞兩端繼續走。

### 1.4 預防
- 出貨前 **含水率達標**（台灣室內 8-12%，梅雨季要除濕）。
- **端封蠟**：粗料端頭塗蠟（端封劑 / anchorseal）防 end check。
- **預留伸縮空間**：桌面與框架用 Z-clip / figure 8 / 木栓孔長條化讓木材橫向能動。
- **拼板年輪交替**（cathedral grain alternating），讓相鄰板互相牽制，降低整體 cup 趨勢。
- **徑切料優先**用在大寬度面板。

## 2. 常見問題 2：板材翹曲（Cup / Bow / Twist）

### 2.1 症狀辨識
| 類型 | 描述 | 圖像 |
|---|---|---|
| **Cup（瓦狀翹）** | 橫向兩邊翹起中間凹，或反之，像瓦片 | 板面像 U 或 ∩ |
| **Bow（弓狀翹）** | 縱向中段翹起或凹陷 | 板面像 ) 或 ( |
| **Twist（扭轉，又叫 wind）** | 兩端對角線翹起，最難救 | 板面像麻花 |
| **Crook（側彎）** | 板的長邊往側面彎（俯視像 ) ） | 縱長方向側彎 |

### 2.2 成因
- **Cup**：徑切弦切混合 / 一面受潮一面乾 / 板太寬沒做拼板。
- **Bow**：堆放時下方有支點不對 / 一面被太陽曬。
- **Twist**：原料本身內應力扭曲 / 木材不對稱含水率變化。最難救是因為 4 個角不在同一平面，沒有單一彎曲軸。
- **Crook**：木紋本身就斜（reaction wood / 應力木）/ 鋸出時就帶。

### 2.3 修補 SOP

#### 輕度 cup（< 3mm 翹度，板寬 300mm 以下）
1. **找出凹面（concave side，較乾的那面）**：把板水平放，凹面朝上。
2. **濕布或噴水**：把凹面均勻打濕，毛巾覆蓋。
3. **重物壓 24-48 小時**：壓平的鐵塊或一疊書，**重點是平均施壓**，不要中間壓邊邊空。
4. 24 小時後檢查，如果還沒平再噴一次。
5. **回到家具濕度環境讓它穩定一週後再用**，不然會反翹。

> 原理：凹面是較乾的那面（fibers shrunk），給水讓它膨回來、和凸面平衡。

#### 中度 cup / bow（3-8mm 翹度）
1. 上述方法可能不夠。
2. **改用刨機 / 壓刨重整一面**：
   - 先過手刨平台（jointer）整一面平。
   - 再過壓刨（planer）整另一面平行。
   - **會耗料 3-5mm**，所以要看板厚有沒有本錢。
3. 板厚剩餘 < 18mm 的話，桌面類就要報廢，當小件用。

#### 嚴重翹曲 / Twist
- **直接報廢，重新拼板**，不要硬救。
- Twist 即使刨平也會繼續扭，因為內應力沒釋放。
- 例外：很薄的板（< 10mm）可以用蒸氣 + 夾具固定 1 週，但成功率 < 50%。

### 2.4 預防
- **年輪交替拼板**：相鄰板的年輪方向相反，cup 趨勢互相抵消。
- **徑切料優先**：徑切（quarter-sawn）橫向變形是弦切（flat-sawn）的 1/2。
- **乾燥到位**再加工：含水率不到 12% 不要拼板。
- **stickers（隔條）堆放**：板與板之間放 1 吋方條，讓空氣流通，避免一面受潮。
- **避免單面塗裝**：塗裝會封水分，一面塗一面沒塗 → 必然 cup。

## 3. 常見問題 3：榫接鬆動（Loose Joints）

### 3.1 症狀辨識
- 椅子坐下會搖。
- 桌子推一推會晃。
- 抽屜面板與側板分離有縫。
- 用力扳能感覺榫頭在榫眼裡晃動。

### 3.2 成因
- **含水率變化**：榫頭縮了、榫眼相對變大。
- **膠合不完整**：當初膠塗不勻、敲入時膠被擠出。
- **長期受力疲勞**：椅子被人坐了 10 年、榫頭被反覆微動磨損。
- **舊膠失效**：PVA 膠 30-50 年後會脆化失效，骨膠（hide glue）反而能撐百年。

### 3.3 修補 SOP

#### Step 1：拆開鬆榫
- **如果膠合已經失敗**，輕敲就開。
- **如果還黏著**，先軟化舊膠：
  - **PVA 膠**：用酒精滲入或熱水（80°C）擠注射器灌入接合線，等 5-10 分鐘軟化。
  - **Hide Glue（骨膠 / 動物膠）**：直接熱水就能溶解。
  - **環氧樹脂**：很難拆，要熱風槍 100°C 軟化。
- **拆下時做記號**：寫編號或貼膠帶，避免裝回去搞錯方向。

#### Step 2：清乾淨舊膠
- **鑿刀刮 + 砂紙包木塊**（**不要直接用砂紙**，會把邊角磨圓影響緊度）。
- 酒精擦拭去除殘膠粉末。
- 完全乾燥後再進行下一步。

#### Step 3：補榫頭（看磨損程度選方案）

##### 方案 A：薄榫頭加薄片（最常用）
- 榫頭頰側（cheek）膠上薄板（**0.5-1mm 木皮 / 牛皮紙 / 修薄的同種木片**）。
- 等乾後刨削到剛好緊塞入榫眼。
- 重新膠合。

##### 方案 B：挖大榫眼補新方榫
- 適合榫眼磨損嚴重。
- 用鑿刀把榫眼擴大到方正。
- 新做一個方木塊膠合進榫眼當「新眼底」。
- 等乾後重新挖出新尺寸的榫眼。

##### 方案 C：整段切除換 Domino / 木釘（最後手段）
- 榫頭已經斷裂或榫眼擴大到無法補。
- 把榫頭完全切除。
- 用 Domino 機 / 圓榫機在兩塊木頭分別挖孔。
- 用 Domino 片 / 木釘 + 環氧樹脂連接。
- **強度通常比原本榫卯弱 20-30%**，不適合椅子主結構。

#### Step 4：重新膠合
| 場景 | 推薦膠 |
|---|---|
| 一般家具 | **PVA Type II（防潮型，Titebond II）** |
| 戶外或衛浴 | **PVA Type III（防水型，Titebond III）** |
| 古董 / 將來可能再修 | **Hide Glue（骨膠）** — 可逆、加水加熱可拆 |
| 縫隙較大 | **環氧樹脂（epoxy）** — 但會永久不可拆 |

> **避免在古董家具上用 Gorilla Glue / 氰基丙烯酸酯（CA）/ 環氧樹脂**，因為下次修的人沒辦法拆。

#### Step 5：夾合 24 小時
- 帶夾（band clamp）/ 棒夾（bar clamp）平均施壓。
- 椅子要 **同時夾合所有方向**（前後左右），否則會偏。

### 3.4 預防
- 膠合前 **先試組**（dry fit），確認所有面都接合到。
- 膠塗 **兩面都塗**（榫頭 + 榫眼），不要只塗一邊。
- 椅子設計時 **牙條 / 橫撐（apron / stretcher）盡量低**，分散應力到所有榫，不要集中到一個。

## 4. 常見問題 4：抽屜變形 / 卡死

### 4.1 症狀辨識
- 拉抽屜要用力。
- 拉一半就卡。
- 推回去推不到底。
- 抽屜底板凸出或下垂。

### 4.2 成因
- **濕度膨脹**：梅雨季實木抽屜膨 1-2mm 就卡。
- **滑軌變形**：木滑軌磨損 / 金屬滑軌歪斜。
- **底板下垂**：薄夾板底承重變形。
- **抽屜面板翹曲**。

### 4.3 修補 SOP

#### 速效（5 分鐘）
- **蠟燭 / 蜂蠟 / 石蠟塊抹滑邊**：抽屜側板下緣、滑軌頂面、櫃體內側摩擦面都抹。
- 推拉幾次讓蠟均勻分布。
- 矽油噴霧（silicone spray）也行，但會讓表面油，不建議在木材上長期用。

#### 永久解（需拆抽屜）
1. **先確認哪邊卡**：抽屜放半開時用紙片塞縫，看哪邊摩擦。
2. **刨削側板與抽屜面板間隙到 1mm**（梅雨季要留到 1.5mm，乾季 0.5mm 就夠）。
3. 用手刨或砂塊修：**只修卡的那一面，其他面不動**。
4. 重新抹蠟。

#### 升級（換金屬滑軌，best）
- 木滑軌換 **三節滑軌（full-extension drawer slides）** 或 **隱藏式底滑軌（undermount slides，Blum Tandem）**。
- 木滑軌慢慢淘汰，但古董家具還是要保留原工法。

### 4.4 預防
- 抽屜面板與櫃體內側 **預留 2mm 上下、1mm 左右間隙**（台灣建議）。
- 抽屜底板用 **5mm 以上夾板**或內加中橫撐（drawer divider）。
- 大抽屜直接用金屬滑軌，不要硬做木滑軌。

## 5. 常見問題 5：椅子搖晃 / 鬆

### 5.1 症狀辨識
- 4 隻腳到地不平（搖搖椅化）。
- 牙條與腳的榫接會晃。
- 後腳與椅背的接合處最常先壞。

### 5.2 成因
- 牙條 / 橫撐榫接全面鬆動（**通常不只一個榫，是 4-8 個榫一起鬆**）。
- 4 隻腳磨損不一樣，某隻磨短了。

### 5.3 修補 SOP（黃金原則：**全拆全膠，不可只修一個**）

> **重點**：椅子 4 隻腳 + 牙條 + 橫撐是一個 3D 結構。如果只膠一個榫，其他鬆榫會把應力拉到剛膠好的那個，幾個月後新膠的會先壞。

#### Step 1：標記所有部件
- 每隻腳寫 1-2-3-4，每根牙條寫對應位置（例如「1-2 牙條」）。
- 拍照記錄。

#### Step 2：全部拆解
- 死膠槌（dead-blow hammer）敲鬆，順著榫的方向敲。
- 拆下 4 隻腳、所有牙條、所有橫撐。

#### Step 3：清舊膠
- 鑿刀 + 砂紙包木塊清乾淨。
- 酒精擦。

#### Step 4：補榫（如果鬆得嚴重）
- 參考第 3 節 SOP（薄片補榫頰側）。

#### Step 5：**同時膠合所有榫**
- 所有榫眼 + 榫頭塗 PVA Type II。
- **一口氣組裝**（PVA 開放時間約 5-10 分鐘，要快）。
- 用 **帶夾（band clamp）繞椅腳一圈** 平均施壓。
- 用 **直角規（square）**檢查椅子方正、4 腳同時著地。
- 24 小時硬化。

#### Step 6：4 腳找平
- 椅子放平整地面，看哪隻腳沒著地。
- 在最長的那隻腳的腳底用砂紙磨掉 1-2mm，反覆測試。
- **不可以墊高短腳**（會讓榫接受力不均）。

### 5.4 預防
- 椅子設計加 **stretcher（橫撐）**串連 4 腳。
- 牙條設計成 **可拆螺絲 + 木栓塞蓋**，將來方便維修。

## 6. 常見問題 6：桌面熱痕 / 水痕

### 6.1 症狀辨識
| 痕跡 | 顏色 | 位置 | 機制 |
|---|---|---|---|
| **白霧（white blooming）** | 白色霧狀 | 塗層下、表面 | 濕氣鎖在塗層下，水分子撐開漆膜 |
| **白色水環（white water ring）** | 白色圓環 | 塗層內 | 杯墊濕氣滲入塗層 |
| **黑色水漬（black water stain）** | 黑色斑塊 | 木材本體 | 水分 + 鐵離子 + 單寧反應（橡木最常見） |
| **燙痕（heat mark）** | 白色或黃色變色 | 表面 | 高溫破壞塗層 / 蒸發水分留下印記 |

### 6.2 修補 SOP

#### 白霧 / 白色水環（漆膜內水分）→ 把水趕出來
**方法 A：吹風機（最安全）**
- 距離 15cm，中段熱風（**不要最熱，會起泡**）。
- 來回吹幾分鐘，看著白霧消失。
- 這個方法適合 **油性 PU、Lacquer（噴漆）**。

**方法 B：熨斗（更強，但風險高）**
- 痕跡上蓋一條棉布。
- 熨斗最低溫，**乾熨不加水**。
- 來回燙 10-15 秒，掀開檢查。
- 短促燙、看狀況、再燙。

**方法 C：油類（緩慢但溫和）**
- 美乃滋 / 凡士林 / 礦物油塗厚一層，靜置 2 小時～整夜。
- 油分子滲入塗層孔隙取代水分。

**方法 D：化學品（專業級）**
- **Mohawk No-Blush Plus Retarder**（噴罐）：噴一下走人，隔天就消。
- 油性 PU / Lacquer 用 **0000# 鋼絲絨 + 礦物油**順木紋輕擦。

#### 黑色水漬（深入木材）→ 草酸（Oxalic Acid）漂白
1. **先去除塗層**（必須露出原木，否則草酸吃不到漬）。
2. 草酸結晶用熱水溶解（**飽和溶液**：100ml 熱水 + 10g 草酸）。
3. 戴 **手套 + 護目鏡 + 口罩**（草酸刺激皮膚與呼吸道）。
4. 用刷子刷在黑漬上，**整片刷不要只刷黑點**（避免色差）。
5. 等 10-30 分鐘，黑漬會逐漸消失。
6. 用清水擦拭中和，等乾。
7. **重新染色 + 重塗**。

> 草酸在台灣化工行（南港 / 三重）或藥房可買到。

#### 燙痕
- 輕度：**白霧處理法**先試。
- 深度：**砂磨 → 補染色 → 重塗**。

### 6.3 預防
- 桌面 **永遠用杯墊 + 鍋墊**。
- 油性 PU 或 Lacquer 比水性 PU 抗熱抗水。
- 客戶交付時附 **保養手冊**寫清楚禁忌。

## 7. 常見問題 7：刮痕

### 7.1 修補 SOP（依深度分級）

#### 淺刮痕（沒穿透塗層）
- **礦物油 + 蜂蠟**：擦一次，光線下看不到。
- **家具修復筆（Minwax Stain Marker / Howard Restor-A-Finish）**：選顏色相近塗一筆，多餘擦掉。
- **核桃仁**：剝開核桃用油性面磨刮痕，天然油脂染色（古早法）。

#### 中度刮痕（穿透塗層）
1. 砂磨 320# 把刮痕周圍打毛（不要打太大範圍）。
2. **補染色**（用棉布蘸著塗，分次薄塗）。
3. 等乾。
4. **重塗保護層**（噴漆 / 刷漆 / 油蠟）。

#### 深度刮痕（傷到木材）
- **小範圍（< 5mm）**：
  - 用環氧樹脂 + 木屑填補（參考第 1.3 節方案 B）。
  - 砂磨齊平 + 補色 + 重塗。
- **大範圍**：
  - **蝴蝶榫補丁式**（參考第 1.3 節方案 C），把蝴蝶榫當裝飾性補丁。
  - **鑿木補嵌（Dutchman patch）**：鑿出方形或不規則凹槽，塞同種木材片進去。

### 7.2 預防
- 桌面玻璃 / 軟墊。
- 椅腳貼氈墊。
- 移動家具用毯子墊著拖。

## 8. 修補工具與材料清單

### 8.1 膠
| 材料 | 用途 | 台灣品牌 / 來源 |
|---|---|---|
| **CA 膠（瞬間膠）** | 微裂、小縫填補 | Starbond（蝦皮）、3M、優力膠 |
| **環氧樹脂（epoxy）** | 中大裂、結構修補 | **紅蝙蝠**（化工行常見）、West System（進口） |
| **PVA Type II** | 一般榫接、防潮 | Titebond II（弘揚 / 蝦皮） |
| **PVA Type III** | 防水 | Titebond III |
| **Hide Glue（骨膠 / 動物膠）** | 古董修補、可逆 | Old Brown Glue（進口）、自熬骨膠 |

### 8.2 填補劑
- **木屑（同種木材磨自）**：最佳對色，在電木刨機集塵桶撿。
- **市售木屑膠**（Famowood / Minwax Wood Putty）：方便但對色差。
- **Bondo（汽車補土）**：超大缺角用，硬度高但醜。

### 8.3 化學品
- **草酸（oxalic acid）**：化工行 / 藥房。
- **酒精（denatured alcohol）**：化工行，95% 以上。
- **熱水**：軟化 hide glue。

### 8.4 修飾用品
- **0000# 鋼絲絨**（最細）：去白霧、磨水痕。
- **家具修復筆**：Minwax Stain Marker / Howard Restor-A-Finish 棒。
- **染色匹配卡片**：自己做色卡，每種木材染色刷在同種木材上對色。
- **打蠟**：Howard Feed-N-Wax（綜合保養）、蜂蠟 + 礦物油（自調）。

### 8.5 工具
- **修邊機（trimmer / palm router）**：銑蝴蝶榫眼。
- **注射器（syringe）**：灌膠。
- **熱風槍 / 吹風機**：軟化舊膠、處理白霧。
- **死膠槌（dead-blow hammer）**：拆椅子。
- **帶夾（band clamp）**：椅子四方膠合。
- **直角規 + 水平儀**：椅子找方正、桌面找平。

## 9. 客戶溝通模板（給木頭仁工作室實用）

### 9.1 收到客訴的標準流程
1. **第一步：安撫**
   > 「先別擔心，木頭家具會有狀況很正常，我先了解一下情況看怎麼幫您處理。」

2. **第二步：診斷（請客戶傳 4 樣）**
   - 整體照（看尺寸 / 位置）
   - 問題特寫（裂縫 / 翹度 / 鬆處）
   - 環境照（家具放哪、旁邊有沒有冷氣 / 暖爐 / 落地窗）
   - 室內溫濕度計讀數（請客戶買一個 NT$200 的溫濕度計）

3. **第三步：估價**
   - 材料：環氧樹脂 / 漆 / 木料費
   - 工：時數 × 時薪
   - 來回運費（基隆 → 客戶家）
   - **不可承諾「修到像新的」**，要說「能恢復到 X 程度，視原因可能會留淡痕」。

4. **第四步：約取件 / 到府**
   - 大件到府、小件取回工作室。
   - 雙方拍照存證（避免後續責任歸屬爭議）。

### 9.2 出貨前文件（預防客訴）
**保固範圍寫清楚：**
- **保固 1 年**：膠合失效、結構性瑕疵、塗層剝落（非人為）。
- **不在保固內**：
  - 含水率變化導致的輕微開裂 / 翹曲（屬自然現象）
  - 人為刮痕、撞擊、水痕
  - 客戶私自移動或改造
  - 環境極端變化（冷氣直吹 / 落地窗西曬 / RH < 40% 或 > 80%）

**附文件：**
- **含水率紀錄**：出貨時測量 + 簽名（自證材料合格）。
- **保養 SOP**：禁忌、清潔方式、定期上蠟頻率。
- **環境建議**：RH 50-65%、避免直射陽光、距冷暖氣 > 1m。

### 9.3 收費 / 不收費判斷
| 情況 | 處理 |
|---|---|
| 出貨 1 個月內、自然瑕疵 | **免費保固** |
| 1 年內、自然瑕疵 | **免費保固** |
| 1 年內、人為損壞 | **酌收材料 + 工**（建議報友情價） |
| 1 年外、任何原因 | **正常收費** |
| 不是自家做的家具 | **正常收費 + 評估** |

## 10. 預防策略（給設計階段）

### 10.1 結構設計
- **浮動面板（floating panel）**：實木桌面與框架交接，**不上膠 + 用 Z-clip / figure 8 固定鐵片**讓面板能橫向呼吸。
- **預留伸縮槽**：面板嵌入框架的溝槽要比面板寬 5mm（橫向）。
- **燕尾不上膠（dovetail floating panel）**：嵌板與框架的燕尾接，膠只塗中央 30mm，兩端不上。
- **拼板年輪交替**：每片相鄰板年輪方向相反。

### 10.2 材料控制
- **含水率達標**：台灣室內目標 **8-12%**，梅雨季要除濕到 12% 以下。
- **徑切料優先**：大寬度面板用徑切（quarter-sawn）穩定。
- **避免應力木**：節疤旁、樹幹彎處的料容易翹。

### 10.3 加工流程
- **粗料 → 靜置 1 週 → 細刨**：讓內應力釋放，不要剛開鋸就最終尺寸。
- **端封蠟**：粗料端頭塗端封劑（Anchorseal / 蜂蠟）防 end check。
- **stickers 堆放**：板與板之間放 1 吋方條讓空氣循環。

### 10.4 塗裝
- **兩面都塗**：木材兩面同時塗裝，避免單面密封造成 cup。
- **油性 PU > 水性 PU 的耐久度**（但水性環保）。
- **桌面用油蠟（hard wax oil）**：好維護、刮痕容易補。

## 11. 何時該放棄（誠實面對）

### 11.1 結構性問題
- **嚴重蟲蛀**：看到 **沙狀 frass（粉末蟲糞）+ 圓孔（< 2mm）= 粉蠹蟲（powderpost beetle）**或家具甲蟲。
  - 處理：燻蒸（fumigation）/ 冷凍 -20°C 一週 / 加熱 60°C 24 小時。
  - **白蟻**（會看到泥道）：直接丟棄 + 找除蟲公司處理整個空間。
  - 嚴重蛀蝕已經影響結構：丟棄。

- **碳化 / 火災後**：木材纖維已經破壞，不修。
- **多次修補處再壞**：**過度修補的木材像被縫了 100 針的衣服**，再縫也撐不住。

### 11.2 經濟性判斷（成本比）
| 修補成本 / 重做成本 | 建議 |
|---|---|
| < 30% | 修 |
| 30-50% | 修，但要評估時間成本 |
| **> 50%** | **重做** |
| > 80% | 重做 + 把舊家具拍照當紀念 |

### 11.3 情感價值例外
- 客戶情感重要的家具（阿公留下、結婚紀念），**就算成本超過 100% 也要修**。
- 但要事先溝通成本，不要修一半才驚訝。
