## 1. 明式家具五大類總覽

王世襄在《明式家具研究》中把明式家具分成 **椅凳、桌案、床榻、櫃架、其他**（含屏風、台架、箱匣）五大類。這個分法已成中國家具學界通用標準。

### 1.1 椅凳類（坐具）

王世襄把坐具再細分為 **杌凳、坐墩、交杌、長凳、椅、寶座** 六小類。

| 小類 | 代表 | 結構特徵 |
|---|---|---|
| 杌凳 | 方杌、長方杌 | 無背無扶手、四足、束腰或無束腰 |
| 坐墩 | 鼓墩、繡墩 | 圓筒形、開光、上下兩道弦紋 |
| 交杌 | 馬札（胡床演變） | X 形可摺、軸釘交接 |
| 長凳 | 二人凳、春凳 | 長度 = 1.5～2.5 倍寬度 |
| 椅 | 燈掛椅、官帽椅、圈椅、玫瑰椅、太師椅 | 有背或扶手 |
| 寶座 | 須彌座式椅 | 體量大、雕飾繁、王座格局 |

**椅子的子分類最重要**（《明式家具研究》上冊頁 33-58 有完整圖解）：

- **燈掛椅**：搭腦兩端出頭翹起、無扶手、輕巧。江南書房常用。
- **官帽椅（四出頭）**：搭腦與扶手皆出頭，形如官帽帽翅。北方常見、體量較大。
- **南官帽椅（不出頭）**：搭腦扶手不出頭、轉角作軟圓角。江南風格。
- **圈椅**：扶手與搭腦連成一條弧線、楔釘榫接 5 段木料。最具書卷氣。
- **玫瑰椅**：靠背扶手等高（皆短）、體量小。明代女子閨房、書房窗下使用。
- **太師椅（清式）**：嚴格說已非明式，清中期出現的雕飾繁複款，與明式風格不同。

### 1.2 桌案類（承具）

王世襄把承具分成 **炕桌類、桌類、案類、几類** 四小類。

**桌與案的根本差別**（這點國外資料常常搞錯）：

- **桌**：四足在板面四角、足與板面用「抱肩榫」或粽角榫直接相接。
- **案**：四足內收、不在四角；足上端用「夾頭榫」或「插肩榫」夾住牙條，板面再蓋在牙條上。

換句話說：「桌」是腿頂著面、「案」是腿夾著牙條再頂面。這個差別決定了榫卯選擇。

代表器形：

- 桌類：八仙桌、半桌、條桌、方桌、書桌、棋桌、酒桌
- 案類：平頭案、翹頭案、架几案、畫案、條案
- 几類：香几、茶几、花几、炕几

### 1.3 床榻類（臥具）

| 小類 | 特徵 |
|---|---|
| 榻（彌勒榻、羅漢床縮小版） | 三面無圍、單人坐臥 |
| 羅漢床 | 三面圍子（後高側低）、可坐可臥 |
| 架子床 | 四柱頂蓋、可掛簾，閨房主床 |
| 拔步床 | 床外加廊、相當於「床中房」，最豪華 |

### 1.4 櫃架類（庋具）

- **架格（書架）**：通透無背板、層板可調或固定
- **亮格櫃**：上層無門（亮格陳設）+ 下層有門（櫃身）
- **圓角櫃**：櫃帽噴出、四足為圓料、門軸用「門臼」結構（上下兩個淺孔軸轉）
- **方角櫃**：四足方料、門軸用合頁
- **頂箱櫃（四件櫃）**：兩個櫃 + 兩個頂箱，組合 4 件，故名

### 1.5 其他類

屏風（地屏、插屏、圍屏）、台架（鏡台、衣架、面盆架、燭台）、箱匣、轎子、文房雜件。

## 2. 明式家具的 module 比例邏輯

王世襄在《明式家具研究》提出明式家具的尺寸**不是隨意定**，而是依「圭臬尺寸」與器物用途相互推導。這套 module 邏輯在歐美傳統家具是沒有的，是中文資料獨有。

### 2.1 椅子的圭臬尺寸（《明式家具研究》上冊頁 90-95）

明式椅子的座面高度遠**低於現代座椅**：

| 部位 | 明式典型 | 現代西式 |
|---|---|---|
| 座面高 | 48~52 cm（一尺五寸） | 42~46 cm |
| 座深 | 45~50 cm | 42~48 cm |
| 座寬 | 50~58 cm | 45~50 cm |
| 扶手高（從座面起算） | 22~25 cm | 18~23 cm |
| 椅背總高 | 100~120 cm（搭腦頂） | 80~100 cm |

**為什麼座這麼高？** 因為明代人**穿袍服、不盤腿坐**（盤腿坐是日本／榻榻米習慣）。袍服需要落腳穩、上半身直立。座高接近膝蓋，小腿垂直、足踏實地。圈椅常配「腳踏」（腳凳），是因為作畫、寫字時可踏上去調整座姿。

**座深淺、椅背高**是明式椅子的核心比例。座深淺逼使坐者**腰部貼椅背**，椅背高頂住肩胛甚至枕骨，整體把人「挺直」起來。這是儒家「正襟危坐」的物質載體。

### 2.2 桌案的長寬高比例

**畫案／書案**（《明式家具研究》上冊頁 116-128）：

- 高度：80~85 cm（明代尺度，與現代書桌相當）
- 案面長：寬 ≈ 2:1（長條案、畫案）或 3:1（極長條案）
- 案腿粗細：邊長 = 案面長度 / 25 ~ 30（即長 200 cm 案、腿邊長約 7~8 cm）

**八仙桌**：
- 邊長 ≈ 高度 × 1.05～1.15（接近正方但略寬）
- 標準八仙桌：邊長 90~95 cm、高 82~85 cm
- 半桌（半張八仙桌）：90×45、靠牆使用

### 2.3 櫃架的高與寬比例

**圓角櫃**：
- 高：寬 ≈ 2.0~2.4 : 1（比西式櫃更瘦高）
- 「側腳」：四足下端外撇、上端收，每邊收 1~2 cm（百分之一左右），讓櫃看起來穩定
- 「噴面」：頂板（櫃帽）四邊比櫃身各噴出 2~3 cm，遮雨水（早期可能放戶外）、視覺上「戴帽」

**架子床**：
- 床面寬：床面長 ≈ 1 : 1.6（接近黃金比）
- 柱高：床面高 ≈ 1.6 : 1（連帳）

### 2.4 線腳的「指甲圓」比例

線腳是明式家具最具識別度的細節。王世襄《明式家具研究》上冊頁 75-89 列了 30 多種線腳剖面，**最常用的是「指甲圓」**：

- 主邊倒成 1/4 圓弧、半徑 = 板厚 / 4 ~ 1/3
- 弧後接一道平直窄面（約 2~3 mm）再接一道小凹槽（陰線）
- 「肥而不腫、瘦而不削」是評線腳的標準

明式線腳工具叫**線刨**（廣東話「平鉋」），刃口形狀就是線腳剖面的反形。一套老師傅自己有 50~100 把不同線形的線刨。

## 3. 經典榫卯詳解

> 這節是這份檔的重點。joinery.md 章節 10 只列了表格、沒拆解。這裡每個榫卯附 **用途／比例／手工 step／常見錯誤**。
>
> 主要參考：阮章魁《木工榫卯》、谷岸《明式家具製作工藝》。

### 3.1 攢邊打槽裝板（最常用，所有桌面案面都用）

**用途**：所有面板（桌面、案面、椅面、櫃門心、抽屜底）都用。把面心板（panel）裝進四邊框（frame）的槽溝裡。

**結構**（阮章魁《木工榫卯》頁 42-50）：
- 四邊框相互用「格肩榫」45° 斜接、單面或雙面
- 四邊框內側挖一道凹槽（槽寬約板厚 1/3、深約板厚 1/2）
- 面心板四邊削薄成「板舌」嵌進槽溝
- **不上膠**！面心板浮在槽中，留 1~2 mm 伸縮空間

**為什麼不上膠？** 中式家具明白實木會四季伸縮（橫紋方向變化最大），把面心板鎖死必裂。攢邊打槽裝板讓板「自由呼吸」。這是中式家具能傳幾百年的關鍵技術之一。

**手工 step**：
1. 邊框料先刨直、定厚
2. 內側拉槽（用槽鉋或凹線刨）
3. 面心板拼板（多用平拼或榫舌邊拼）
4. 板四邊削舌：留中央 1/3 厚度做舌、舌寬 = 槽深 + 1 mm 預留
5. 邊框 45° 斜接、兩端開格肩榫頭
6. 試組、調整面心板舌長使邊框能合攏
7. 邊框膠合（**只膠邊框 45° 接面，不膠面心板**）

**現代問題**：很多 DIY 家具直接把面板釘死或膠死，過幾年就裂。明式工法改良版：用 Domino 替格肩榫、面心板浮裝原理保留。

### 3.2 抱肩榫（束腰桌椅腿足上端標配）

**用途**：束腰家具（桌、凳、椅）的腿足與牙條接合處。

**「束腰」是什麼？** 桌面或座面下方有一道內收的窄帶（像繫腰帶），下面才是牙條與腿。束腰是明式家具最美的部位之一，也是和西式桌最大的視覺差異。

**結構**（谷岸《明式家具製作工藝》頁 78-92）：
- 腿足上端**留出一段方料**（露出在面板上抱住板邊）
- 方料下挖出「束腰段」與「牙條接面」
- 牙條兩端做榫頭、塞入腿足上的榫眼
- 榫眼旁邊有 **45° 斜肩** 切口、咬住牙條角（防牙條向外翻）
- 斜肩內側挖小三角眼、再讓牙條角伸入鎖緊

**比例**：
- 腿足上端方料邊長 = 牙條厚 + 束腰段厚 + 留肩 5~8 mm
- 榫頭厚 = 牙條厚 / 3
- 斜肩 45°、長度 = 牙條厚

**手工 step**：
1. 腿足料先方正、定四面
2. 上端依設計畫線：上面板段、束腰段、牙條段
3. 以鑿刀切除束腰段料（保留兩面與內側、只切前面）
4. 牙條接合面開榫眼（用方鑿）
5. 榫眼外側再切 45° 斜肩
6. 牙條兩端做相應榫頭與斜肩
7. 試組、調整、上膠夾合

**為什麼要斜肩？** 純粹方榫塞入只防上下動，不防外翻。斜肩咬住牙條角，把牙條的橫向應力鎖死。圓桌、八仙桌之所以結構穩定（不需任何金屬五金），抱肩榫是核心。

### 3.3 夾頭榫（案類腿足與牙條的標配）

**用途**：案（畫案、書案、平頭案、翹頭案）的腿足上端夾住牙條。

**結構**（阮章魁《中國家具榫卯結構》頁 102-115）：
- 腿足上端**整段開口**、像 U 字形夾子
- 開口寬 = 牙條厚 + 牙頭厚（兩側都要包）
- 牙條（橫向長條）放進開口、兩端用榫頭塞入腿足內側榫眼
- 牙頭（豎向短條）也插入腿足正面

也就是：腿足上端**從三個方向被夾住**——前面有牙頭、左右有牙條、頂上有案面。應力分散到三方。

**比例**：
- 開口深度 = 案面板厚 + 1~2 mm 預留
- 開口寬度 = 牙條 + 牙頭厚度總和
- 牙條榫頭厚 = 牙條厚 / 3

**手工 step**：
1. 腿足上端畫出三向開口線
2. 用框鋸沿線縱切兩刀
3. 鑿掉中央木料（中間部分整段挖空）
4. 牙條兩端做榫頭、牙頭做明榫
5. 試組：先放牙條進開口、再從正面塞牙頭、最後蓋案面

**為什麼明式案「特別穩」？** 因為夾頭榫把牙條牙頭整圈包住、案面再壓上去，等於把腿足上端「灌封」死。畫案常見長 2 米以上、單足卻細到 6~7 cm 邊，靠的就是這個榫。

### 3.4 插肩榫（案類腿足進階款）

**用途**：高級案類（特別是畫案、條案）的腿足與牙條接合，比夾頭榫更隱形美觀。

**與夾頭榫差別**：
- 夾頭榫是「腿夾牙條」、開口從外側看得到
- 插肩榫是「腿與牙條互嵌、外側齊平」、表面看不到結構

**結構**（阮章魁《中國家具榫卯結構》頁 116-122）：
- 腿足上端外側削成 V 形斜肩
- 牙條對應位置也削成倒 V 形凹口
- 兩者扣合後，外表面齊平、看起來像一條連續曲線
- 內側仍有榫頭與榫眼鎖定

**手工 step**：
1. 腿足上端外側畫 V 形斜肩線（角度約 60°）
2. 鑿出 V 形凸出
3. 牙條對應位置鑿出 V 形凹槽
4. 牙條再鑿榫頭（伸入腿足內側榫眼）
5. 試組、修配（V 形必須緊密無縫，這是門面）

**判定古案是否高級**：看牙條腿足相接處——平整無縫、看不到結構就是插肩榫。看得到 U 字夾頭就是夾頭榫。前者工高、後者實用。

### 3.5 霸王棖（圓桌與某些方桌專用）

**用途**：把方桌或圓桌的桌面背側中心、與四足下方拉住，形成「米」字內撐。

**為什麼叫霸王？** 「霸王舉鼎」之意——一根曲木撐起整個桌面。

**結構**（《明式家具研究》上冊頁 145、阮章魁《木工榫卯》頁 156-162）：
- 上端勾住桌面背面的「掛榫」（鉤形榫，咬住穿帶）
- 下端入腿足內側的榫眼
- 中段為彎曲木（多為一體曲料）
- 共四根，每根接一隻腿

**為什麼用曲料？** 直棖會擋住坐者腳膝、影響使用。曲線拐彎避開人腿空間。這是明式家具「兼顧結構與人體工學」的代表作。

**手工 step**：
1. 找一段**自然彎曲**的料（最佳）或用大料切出曲形
2. 上端做勾形掛榫
3. 下端做方榫
4. 桌面背側開穿帶、穿帶上挖鉤眼
5. 腿足內側開方榫眼
6. 安裝順序：腿先固定 → 從下塞進腿榫眼 → 上端推入掛在穿帶上

### 3.6 粽角榫（方角櫃、書架四角專用）

**用途**：三向木料以 45° 互嵌交於一點（如方角櫃四個上下角）。

**結構**（阮章魁《中國家具榫卯結構》頁 138-145）：
- 三根木料（兩水平 + 一垂直）末端各削 45° 斜肩
- 三個斜肩相交形成 Y 字、交於一點
- 內部各開榫眼、相互穿榫鎖定

外觀像一顆**粽子的角**——三向都是斜面、交於頂點，故名粽角。

**比例**：
- 三料端部各斜削 45°、長度 = 料厚（取等寬最美）
- 內藏榫頭厚度 = 料厚 / 3

**手工 step**：
1. 三料定厚定寬一致
2. 各自端部畫 45° 斜線（兩個方向）
3. 鋸出斜面（複合斜切，現代用斜切鋸最方便）
4. 內部開榫眼與榫頭
5. 試組（三向同時推入是關鍵，順序錯就裝不上）

**現代簡化**：很多現代仿明家具用斜切+暗銷代替粽角榫，外觀像但結構大幅簡化。要看真粽角榫得在博物館。

### 3.7 楔釘榫（圈椅扶手與所有弧形連接專用）

**用途**：兩段弧形木料端對端接合，主要用在圈椅的扶手（一條弧線通常由 5 段木料拼成）、月牙桌的弧邊。

**結構**（《明式家具研究》上冊頁 47、谷岸《明式家具製作工藝》頁 145-160）：
- 兩段弧木的端部各做「搭口」（半搭接、咬合面為長方形）
- 搭口中央各開一個方孔
- 兩孔對齊後從上方插入一根**楔釘**（梯形木栓，一頭粗一頭細）
- 楔釘從上往下打進去，越打越緊，把兩段牢牢鎖在一起

**比例**：
- 搭口長度 = 木料寬 × 1.5~2
- 搭口厚度 = 木料厚 / 2
- 楔釘上寬 = 搭口寬 / 3、下寬略小、長度 = 木料厚

**手工 step**：
1. 弧木兩段各鋸出搭口
2. 搭口中央用方鑿開孔
3. 試組搭口、孔對齊
4. 削楔釘（一頭粗、一頭細、四面平）
5. 用木槌敲入、打到不能再進為止
6. **不上膠**（重點）——楔釘乾縮可換新、活動接合

**為什麼是圈椅核心技術？** 圈椅扶手是一條 270° 弧線、長度近 2 米，但黃花梨原料寬度極限 60~80 cm。古匠人用 5 段木料拼出整條弧、楔釘榫接合處幾乎看不出來，是明式家具最經典的「化整為零」工法。

**現代狀況**：木頭仁如果要做圈椅，扶手用整片彎曲薄板積層（bent lamination）會比楔釘榫容易很多。但要復刻古典就得學楔釘榫。

### 3.8 走馬銷（活拆家具標配）

**用途**：可拆卸接合，用於羅漢床圍子、屏風扇與底座、架子床各部件。

**結構**（阮章魁《中國家具榫卯結構》頁 178-185）：
- 銷釘（公榫）為**燕尾形 + 漸細**（上窄下寬）
- 榫眼（母榫）為**葫蘆形**（一頭大一頭小）
- 銷釘從大孔處插入、推到小孔處卡死
- 拉回大孔處可拔出、不破壞結構

可拆卸的關鍵：銷釘**不上膠**、靠形狀鎖定。

**用途案例**：
- 羅漢床三面圍子可拆下平放收納
- 架子床六根柱子可分件搬運
- 屏風扇可分扇收納

這是中式家具「便於搬遷」的智慧。古代家庭搬家、移動需求大，活拆設計很重要。

### 3.9 銀錠扣（蝴蝶榫，拼板防裂）

**用途**：拼板背面跨在膠線上、防止膠線開裂、或修補既有裂縫。

**結構**：兩端寬、中央窄的雙燕尾木片（形似銀錠或蝴蝶），嵌入拼板背面的淺槽。

**比例**：
- 燕尾角度 1:6 ~ 1:8
- 厚度 = 板厚 / 3（不貫穿）
- 長度 = 跨膠線兩側各 30~50 mm
- 木片紋理**與板材橫紋方向一致**（這是關鍵，紋理錯就無防裂效果）

**注意**：銀錠扣只防「橫向拉開」、不防「彎曲」。所以它通常配膠合或穿帶使用、不單獨承力。

repair_restoration.md 章節 1.3 方案 C 已寫過西式蝴蝶榫的修復應用。明式銀錠扣是裝飾兼功能、可在正面看見（用對比色木種強調）；西式 Dutchman 多藏背面。

### 3.10 穿帶（面板背面標配）

**用途**：所有實木面板（桌面、案面、櫃門心、屏心）背面都加穿帶，**防翹曲、加強結構**。

**結構**（阮章魁《木工榫卯》頁 38-45）：
- 一根橫木（穿帶）跨過面板背面、與面板紋理**垂直**
- 穿帶上端削成燕尾形（窄頂寬底）
- 面板背面挖出對應的燕尾槽
- 從面板側邊滑入（從寬邊往窄邊）、卡緊但不黏
- 面板乾縮時穿帶可微滑、不限制呼吸

**比例**：
- 穿帶寬 = 面板寬 / 8 ~ 1/6
- 穿帶厚 = 面板厚 × 1.2 ~ 1.5（穿帶要比面板厚才有用）
- 穿帶數量：面板長 < 80 cm 用 2 根、80~150 cm 用 3 根、150 cm 以上用 4~5 根

**為什麼不能膠死？** 穿帶與面板紋理垂直、四季漲縮方向相反。膠死必裂。傳統做法只在中央一點打釘或留些 play、其餘讓它自由滑動。

**現代替代**：歐式做法用 figure-8 fastener 或 zip clip 固定面板到框架上、原理一樣（讓面板自由動）。但實木 18~25 mm 厚的長面板，沒有穿帶等於宣告報廢。

### 3.11 其他高級榫卯（簡介）

| 名稱 | 用途 | 重點 |
|---|---|---|
| **格肩榫** | 邊框 45° 接合（門框、面板攢邊） | 角度 45°、肩深 = 料厚 / 3 |
| **悶榫**（暗榫） | 表面看不到的盲榫 | 用在貴重木料、不破壞紋理連續性 |
| **扎榫**（加銷） | 榫頭旁加竹釘或木釘 | 防榫頭鬆動 |
| **獨步榫** | 圓料相接（圓椅腿與牙條） | 圓榫頭直徑 = 圓料 / 2 |
| **羅鍋棖** | 桌椅下方 ∩ 形彎棖 | 中央拱起、避腿空間 |
| **十字棖** | 四足底部 X 形連接 | 加強橫向穩定 |
| **管腳棖** | 椅腿下方一圈橫撐 | 前低後高（人腳前置坐姿用） |

詳細結構圖參考阮章魁《中國家具榫卯結構》，每種都有三視圖。

## 4. 用料邏輯：哪種木做什麼

> wood_species.md 章節 2.12 已列紅木 Janka 表。這節談**書本層級的用料邏輯**——王世襄、鄒華熹的觀察：哪種木做什麼器形最對。

### 4.1 黃花梨（最高級）

**學名**：*Dalbergia odorifera*（海南黃花梨）

**特色**：
- 顏色金黃帶橙、紋理如行雲流水（「鬼臉」紋是頂級特徵）
- 比重 0.82~0.94、Janka 數值文獻不一（多估 2200~2400）
- 含微量香氣（淡淡降香）
- 加工性**極佳**——硬而不脆、刨削光滑、不易撕裂

**最適合做**（《明式家具研究》上冊頁 8-15、《明式家具珍賞》通篇）：
- **圈椅**（弧形扶手要求材質韌、不裂）
- **官帽椅**（搭腦扶手出頭部位需強韌）
- **書桌、畫案**（大尺寸面板要求穩定不翹）
- **圓角櫃**（大面板＋曲料兼具）

**為什麼明代圈椅幾乎都黃花梨？** 因為圈椅扶手要彎、要分段拼、要楔釘榫接——這些都需要**極高韌性 + 不易撕裂**的材性。黃花梨剛剛好。紫檀太脆、雞翅木太疏、鐵力木太澀，都不如黃花梨。

**現代狀況**：海南黃花梨已**幾近絕跡**（2014 年被列入 CITES II），正規市場極少見。所謂「越南黃花梨」（*D. tonkinensis*）紋理略遜、價格也已暴漲。**真品價格每公斤可達數萬人民幣**。

### 4.2 紫檀（最莊重）

**學名**：*Pterocarpus santalinus*（小葉紫檀，最高級）

**特色**：
- 顏色深紫近黑、新切偏紅、氧化後轉深褐
- 比重 1.05~1.26（**入水即沉**）
- Janka 2700+
- 含油脂、表面打磨後**自帶光澤**（不需上漆）
- 加工性**差**——硬、脆、刀具磨損極快

**最適合做**（《明式家具研究》上冊頁 18-24）：
- **寶座、太師椅**（體量大、雕飾多、要威儀感）
- **頂箱櫃**（高大、需穩重感）
- **小件雕刻**（紫檀拋光後光澤如玉）

**為什麼紫檀不做圈椅？** 太脆。圈椅弧形受力大、紫檀容易順紋撕裂。

**為什麼紫檀適合雕刻？** 因為密度極高、雕刻細節能保留、不易崩線。

**清宮造辦處**幾乎所有重要家具都用紫檀（特別是雍正、乾隆兩朝），形成「清式紫檀風」、與「明式黃花梨風」對比。

### 4.3 雞翅木（最有特色）

**學名**：*Millettia laurentii*（非洲雞翅）、*Cassia siamea*（鐵刀木）等

**特色**：
- 顏色黑黃條紋相間、紋理形如雞翅羽毛
- 比重 0.7~0.95、Janka 1700~2200
- 紋理**極具裝飾性**——本身就是花紋，不需雕飾
- 加工性中等——不如黃花梨光滑、易出毛刺

**最適合做**（鄒華熹《中國傳統家具用材》相關章節）：
- **嵌板**（如圓角櫃門心、屏風心）
- **小件案頭文房**（筆筒、鎮紙、印盒）
- **椅背心板**（一塊大紋雞翅木做背板，極有韻味）

**為什麼不做大件？** 因為紋理太搶眼、做整件家具會視覺混亂。雞翅木最適合「點綴」、與其他素面木料搭配。

### 4.4 鐵力木（最堅韌）

**學名**：*Mesua ferrea* / *Hopea* spp.

**特色**：
- 顏色深褐、紋理粗
- 比重 1.0+、Janka > 3000（與紫檀同級）
- **耐水性極強**、不易蟲蛀
- 加工性**差**——澀、不光滑、不易刨削

**最適合做**：
- **大案大桌的腿足**（需粗料、抗壓）
- **戶外亭台、舟船**
- **粗實的庫房櫃架**

**為什麼少做精細家具？** 因為刨削後紋理粗、不夠光滑、難打磨拋光。鐵力木走的是「實用粗獷」路線、與黃花梨「文雅精緻」路線剛好相反。

### 4.5 軟木（柏、松、杉）

明式家具不只用紅木。地方家具、平民家具大量用**軟木**：

- **柏木**（*Cupressus*）：山西、陜西常用、做柏木櫃
- **杉木**（*Cunninghamia*）：江南做箱、抽屜
- **松木**：北方家具骨架、大件粗料

王世襄在《明式家具研究》專章提到「**柴木家具**」（軟木雜木家具）、認為其工藝美學與紅木家具同樣值得尊重。**不要把明式家具等同於紅木家具**——這是收藏圈的誤解。

### 4.6 木頭仁台灣對應替代

| 古典紅木 | 台灣可得替代 | 替代理由 |
|---|---|---|
| 黃花梨 | **台灣肖楠**（韌、香、紋黃） | 加工性類似、有香氣、本土 |
| 紫檀 | **台灣紅檜古料** 或 **印度玫瑰木** | 莊重感類似、紅檜有光澤 |
| 雞翅木 | **沙比利** 或 **斑馬木** | 紋理裝飾性類似 |
| 鐵力木 | **柚木** 或 **白橡** | 耐水耐用、可做戶外 |
| 柏木 | **台灣扁柏** | 直接對應、本土最佳 |

木頭仁的觀點：做明式形制不一定要紅木。**用台灣肖楠做圈椅**比追海南黃花梨實際得多——肖楠韌性夠、有香氣、本土合法、價格友善。重點是**形制與比例對**、用料是次要。

## 5. 雕飾與紋飾（淺說）

> 木頭仁主要做**素面家具**（明式講究「以木為美、雕飾為輔」），這節只簡介書中提到的紋飾體系。

### 5.1 明式雕飾的克制原則

王世襄在《明式家具研究》上冊頁 85 強調：「**明式之美在素**」。明代上品家具雕飾極節制——多數只在**牙條、券口、靠背板**等「**面積有限的關鍵部位**」做小幅雕飾，整體仍以素面為主。

清式則完全相反：滿身雕飾、不留素地。

明式三大雕法：
- **淺浮雕**（最常見）：紋路凸起 1~3 mm、不見刀痕、靠陰陽變化營造立體感
- **透雕**（券口、靠背板）：把紋理之間的木料完全鏤空、形成花窗效果
- **陰刻**（線刻）：刀刻線條、不去料、僅留紋路

### 5.2 常見紋樣（《明式家具珍賞》圖鑑通篇）

| 紋樣 | 寓意 | 常見部位 |
|---|---|---|
| 雲紋 | 祥雲、富貴 | 牙條、券口 |
| 螭紋（小龍） | 權勢、文雅 | 靠背板、抽屜面 |
| 卷草紋 | 連綿不絕 | 邊框、線腳 |
| 如意紋 | 如意吉祥 | 角部、結合處 |
| 開光（菱形/圓形邊框） | 框景、聚焦 | 大面板中央 |
| 拐子紋（直角折線） | 規矩、嚴謹 | 線腳、邊框 |

### 5.3 螺鈿、嵌銀（少數高級件）

- **螺鈿**：把貝殼磨成薄片、嵌入木材（多嵌黑檀、紫檀）。明式用得不多、清式宮廷大量用。
- **嵌銀**：銀絲嵌入凹槽、形成銀亮線條。多見於文房小件（如筆筒、印盒）。

這兩種工藝**不在木頭仁主業範圍**、提到讓 bot 知道有這個東西即可。

## 6. 對台灣現代木工的啟發

### 6.1 仍實用的傳統工法

這些傳統工法**今天仍該學**、不只是復古：

1. **攢邊打槽裝板**——所有實木大面板的標準解。比直接拼板膠死可靠 10 倍。木頭仁設計器的桌面、櫃門心都該用這個原理。
2. **穿帶**——任何大面板（> 80 cm）都該加穿帶。是防翹曲的物理規則、不是裝飾。
3. **不上膠的活動接合**——桌面與框架、面板與穿帶、楔釘榫等。讓木材自由呼吸是傳家家具的關鍵。
4. **抱肩榫的斜肩咬合**——任何 T 字接合（牙條入腿）斜肩比平肩強多倍。Domino 加 45° 二次切口可模擬。
5. **線腳（指甲圓 + 陰線）**——讓邊角不刺手、視覺柔和。比西式單純倒角優雅。

### 6.2 只剩裝飾意義的工法

這些可學可不學、現代有更好替代：

1. **粽角榫**——三向 45° 互嵌極難手工，現代用斜切 + Domino 結構等強。
2. **楔釘榫**——複雜、易出錯。圈椅扶手用 bent lamination（彎曲積層）效果一樣甚至更穩。
3. **雕飾紋樣**——除非客戶特別要、否則明式風格的素面更現代。
4. **走馬銷**——古代搬家用、現代家具搬一次定位很少拆。

### 6.3 木頭仁的應用建議

- **拍片題材**：「攢邊打槽裝板」「穿帶為什麼必要」「明式椅子座高為什麼是 50 cm」這幾個都是好題、且**外國木工頻道沒人講**，差異化內容。
- **教學課程**：「明式邊框攢框」可作為中級榫卯課（介於指接燕尾與全手工之間）。
- **設計器擴充**：bot 提到的 module 比例（如椅高 50 cm、案高 80 cm、抱肩榫斜肩 45°）可以放進 wooden-ren-designer 的明式家具模板。
