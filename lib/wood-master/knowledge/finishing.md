## 第 0 章 安全第一 — 油布自燃（必讀！每年都有工作室火災）

> **這一段如果你只記得一件事，請記住：含油塗料（亞麻仁油、桐油、丹麥油、Tru-Oil、Osmo、Watco、Tried & True、任何 BLO/Tung-Oil-based）擦完的布、棉花、紙巾，揉成一團丟垃圾桶 = 三小時內可能自燃。**

### 為什麼會自燃？
亞麻仁油、桐油等「乾燥油（drying oil）」是靠**氧化反應**硬化的，這個反應本身會放熱。當油布揉成一團、塞進垃圾桶、堆在角落時，熱量散不掉、累積到布料燃點（棉布大約 120°C，紙巾更低），**不需要任何外部火源**就會起火。

美國 NFPA（國家防火協會）統計：每年約 900 起火災由油布自燃造成、1700 起房屋火災由自燃／化學反應造成。ABC News 實測：擦了亞麻仁油的布堆，**3 小時 12 分**自燃起火。

### 正確處理油布（任選一種）
1. **攤開晾乾**：把布攤開掛在金屬欄杆、不可燃地面、或晾衣架上，**單片不重疊**，曬到乾透（24 小時以上、用手摸感覺不到油），再丟一般垃圾。
2. **泡水**：用一個金屬蓋密封罐（餅乾鐵盒最好）裝水，把所有油布丟進去**完全淹過水面**，蓋緊。等收滿一罐再連水帶布丟掉。
3. **絕對不要**：揉成一團丟塑膠垃圾桶、塞進工具櫃、堆在工作台、丟洗衣籃、或丟進有其他可燃物的桶子裡。

> **木頭仁工作室建議流程**：擦完油的廢布，當天**立刻**攤在戶外鐵架上、壓塊磚、隔天再丟。下班前繞工作室一圈，確認沒有任何擦油廢布塞在角落。

## 第 1 章 塗裝前處理

塗裝 70% 的成敗在底材處理。砂磨不到位，再貴的塗料都救不回來。

### 1.1 砂磨號數順序

| 階段 | 號數 | 目的 |
|------|------|------|
| 粗磨 | 80（嚴重缺陷才用）／100 | 削平鋸痕、刨痕、去舊漆 |
| 中磨 | 120 | 一般起手號 |
| 細磨 | 180 | 染色前止步點（軟木 / 開孔木） |
| 精磨 | 220 / 240 | 油性塗料、染色前可選 |
| 超精磨 | 320 | 透明塗料、堅木、塗層之間 |
| 拋磨 | 400+ | 蟲膠 / 樂器 / french polish |

**鐵則 1：不可跳號。** 80 → 180 跳一級，會留下 80 號的刮痕（即便你看不到）。塗上塗料後刮痕會被「拉出來」、看起來像舊家具。每一級最多升 50%（80→120→180→240→320），這是 Norton 與 FineWoodworking 的共識。

**鐵則 2：軟木止於 180-220。** 松木、雲杉、杉木砂到 320 反而**封孔**，染色吃不進去，會變成花斑。

**鐵則 3：堅木看後續塗料。**
- 油性染色 → 止於 150-180（讓染料有空間滲入）
- 水性染色 → 止於 180-220
- 透明塗料（油、蟲膠、PU）→ 220-320 都可，越細越平

### 1.2 不同木材最終研磨號數建議

| 木材 | 最終號數 | 備註 |
|------|----------|------|
| 松木 | 180-220 | 軟木易吃染料，砂太細會花 |
| 檜木（台灣常用） | 220 | 油性表面、不過度砂 |
| 橡木 | 150-180 | 開孔大、染色前必須留空間 |
| 楓木 | 220 | 易花染、配合 pre-stain conditioner |
| 櫻桃 | 220 | 同楓木邏輯 |
| 胡桃木 | 180-220 | 開孔中等 |
| 山毛櫸 / 樺木 | 180-220 | 細孔堅木 |
| 紫檀 / 花梨 | 320-400 | 含油堅木，砂越細手感越好 |

### 1.3 端紋處理（Endgrain Sealing）

端紋（cross-grain / endgrain）是木材**纖維被切斷**的截面，吸收能力大約是面紋的 **3 倍**。如果你不先封端紋，染色時端紋會吃進大量染料、變黑黑一條，醜不可言。

**封端紋常用方法：**
- 木工膠（Titebond、白膠）稀釋 1:1 抹一層、乾後砂掉
- 第一層稀釋過的塗料（PU 用松節油稀釋 50%）
- 蟲膠 1lb cut 一層當 sealer（最好用、不影響後續染色）
- Charles Neil 端紋封閉劑、Titebond II 稀釋

### 1.4 升毛（Raise Grain）— 水性塗料必做

水性塗料（水性染色、水性 PU、Aqua Coat）一上去，水會讓木材纖維**站起來**，乾後變毛糙刺刺。對策叫**預升毛**：
1. 砂到目標號數（例如 220）。
2. 用乾淨抹布沾水、擦遍整個工件（不是泡水）。
3. 等 30-60 分鐘乾透、或吹風機吹乾。
4. 用**同一號數**的砂紙輕輕磨掉站起來的毛刺（3-5 秒就好，不要磨太多）。
5. 上水性塗料 — 木材已經「免疫」、不會再升毛。

油性塗料不需要升毛（油不會讓纖維膨脹），但做了也沒壞處。

## 第 2 章 木紋處理（Grain Filler / 補土 / 蟲洞填補）

### 2.1 為什麼要 grain filler？

橡木、白蠟木、胡桃木、紅木這類**開孔木**（open-pore wood），毛孔大到上完十層 PU 都還是凹凸。如果你想要**鏡面光滑**（樂器、高級家具、漆器風），就要用 grain filler 把毛孔填平。

如果你想要**保留木紋觸感**（北歐風、原木風），就**不要**用 filler，直接上油或薄塗就好。

### 2.2 染色 vs 不染色 filler

- **不染色 / 透明 filler（如 Aqua Coat Clear）**：填平毛孔但不改變木材顏色。後續可染色或不染色。
- **染色 filler（dark walnut, mahogany 色號）**：填的同時把毛孔染成深色，木紋會跳出來、有「腐朽古木」效果。經典橡木家具這樣做。

### 2.3 台灣常用 filler

| 產品 | 類型 | 特性 |
|------|------|------|
| Aqua Coat Clear Grain Filler | 水性透明 | 蝦皮代購可得、低 VOC、好砂、可染色 |
| 補蘇士 / Bondo 木材補土 | 油性 | 補蟲洞、節疤大缺口、硬度高 |
| 木性堂銷售之水性填縫劑 | 水性 | 台灣木工網購最方便 |
| 兩液環氧樹脂（AB 膠）+ 木屑 | 環氧 | 補大洞最強、可染色、硬度高 |
| CA 膠（瞬間膠）+ 木屑 | 瞬間 | 補小蟲洞最快、樂器師最愛 |

### 2.4 Aqua Coat 水性 filler SOP

1. 砂到 220、清潔。
2. 用塑膠刮刀（信用卡也行）**斜對木紋方向**把 filler 抹進毛孔，多塗一點沒關係。
3. 立刻**順木紋方向**用刮刀把多餘的刮掉。
4. 等 45-60 分鐘乾。
5. 用 320 號砂紙**輕輕**磨平（過度砂會把孔裡的 filler 也挖出來）。
6. 重複 2-3 次直到毛孔填平（用手摸不到凹陷）。
7. 才能上染色或塗料。

### 2.5 補蟲洞、節疤洞

- **小洞（< 2mm）**：CA 膠（瞬間膠）滴一滴 + 同色木屑壓進去 → 砂掉。1 分鐘搞定。
- **中洞（2-10mm）**：兩液環氧樹脂（AB 膠）混色粉或木屑、灌進去、等 24h、刨平砂掉。
- **大洞 / 結節崩裂（> 10mm）**：環氧樹脂 + 染色 / 螢光粉做「樹瘤效果」，這是現在很流行的 epoxy river 風。

## 第 3 章 染色（Stain）— Pigment vs Dye

### 3.1 染料 vs 顏料 — 核心區別

| 項目 | 顏料染色（Pigment Stain） | 染料染色（Dye Stain） |
|------|---------------------------|------------------------|
| 本質 | 細小固體粉末，懸浮在介質中 | 可溶解在介質中的分子（鹽類） |
| 著色機制 | 卡進毛孔、不滲入纖維 | 滲入木材纖維、染整個細胞 |
| 透明度 | 較不透明（會蓋紋路） | 高透明（紋路更跳） |
| 抗光 | 抗紫外線好 | 易褪色（染料對 UV 敏感） |
| 對開孔木 | 開孔處變深、紋路強 | 整體均勻染色 |
| 代表品牌 | Minwax、米諾佳 | TransTint、TransFast、Mixol |

**白話**：
- 想要「強調紋路」（橡木、胡桃木）→ pigment stain。
- 想要「整體變色但保留紋路」（楓木、櫻桃）→ dye stain。
- 兩個一起用：先 dye 染整體底色、再 pigment stain 強化毛孔，這是高階家具廠的標準作法。

### 3.2 主要染色類型

#### 油性染色（Oil-based Stain）
- 代表：Minwax Wood Finish、米諾佳、Varathane Premium
- 滲透深、易控制、易擦掉多餘
- 乾燥慢（4-8 小時）、需通風
- 上面要等 24 小時才能上 PU
- **木頭仁推薦**：新手、家具、深色木選這個

#### 水性染色（Water-based Stain）
- 代表：General Finishes Water-Based Stain、Saman、Minwax Water-Based
- 環保、低 VOC、清水洗手
- **會升毛**（必須先 raise grain）
- 乾得快（2-4 小時）
- 顏色乾後比濕的淺

#### 凝膠染色（Gel Stain）
- 代表：General Finishes Gel Stain、Minwax Gel Stain
- **不滲入**、像漿糊一樣坐在表面
- 適合**難染木**（楓、櫻桃、松、MDF、舊漆面）
- 可疊色、可不染端紋、可垂直面不流掛
- 不需要 pre-stain conditioner（因為不滲入、不會花）

#### 預先處理（Pre-stain Conditioner）
- 適用：松木、楓木、櫻桃、樺木（軟木 + 易花染木）
- 原理：先用稀釋的封閉劑「半封」毛孔，避免染料只往低密度區跑
- SOP：擦一層 → 等 5-15 分鐘 → 不擦掉 → 直接染色（不要等乾透）
- **重要**：用 dye stain 時**不要用** pre-stain conditioner（會讓 dye 上不去、顏色變淡）

### 3.3 標準染色 SOP（油性 / 水性 wiping stain）

1. 砂磨到目標號數、清潔（壓縮空氣 + 黏塵布）。
2. （如需要）擦 pre-stain conditioner、等 5-15 分鐘。
3. 用棉布、海綿、或刷子**抹一層染料**，順木紋。
4. 等 **5-15 分鐘**（看產品標示，溫度低就久一點）。
5. **乾淨棉布順木紋擦掉多餘染料**（這一步決定深淺）。
6. 等 24 小時乾透。
7. 顏色不夠深？再染一次（重複 3-6）。
8. 確認乾透後，才能上 sealer 或 PU。

### 3.4 天然染色（DIY 風）

| 材料 | 顏色 | 備註 |
|------|------|------|
| 核桃殼水（煮沸） | 暖棕 | 古法、需多層、易褪色 |
| 紅茶 / 黑咖啡 | 淺棕 | 配合鐵醋液變黑 |
| 鐵醋液（鐵釘泡白醋） | 灰黑 | 反應木材單寧、橡木變灰 / 胡桃變黑 |
| 高錳酸鉀（漂白劑） | 棕紫 | 危險品、需手套眼鏡 |
| 紫蘇 / 洛神 | 紫紅 | 短期、易褪 |

天然染色適合 DIY 風格、IG 美感家具，不適合長期使用 / 客戶交付（顏色不穩定）。

## 第 4 章 油性塗料（Oil Finishes）

### 4.1 純亞麻仁油（Boiled Linseed Oil, BLO）

- 來源：亞麻籽榨油 + 加熱聚合（讓乾燥變快）。**「Boiled」指的是聚合，不是真的煮過**。
- 優點：滲入深、突顯木紋、便宜、好取得
- 缺點：**防護差**（不防水、不防熱、不耐刮）、需定期保養（半年一次）、會慢慢變黃
- 用途：工具柄、農具、復古家具、底油（後續再上其他）
- 乾燥：24-72 小時 / 層
- **絕對警告**：油布**強烈自燃傾向**，亞麻仁油是所有油裡自燃風險最高的。

### 4.2 桐油（Tung Oil）

- 來源：油桐樹果實榨油（中國古法、油紙傘也用這個）
- 優點：防水比 BLO 好、不變黃（亞麻仁會黃、桐油不黃）、食品安全（純桐油、非聚合過的）
- 缺點：**乾得超慢**（純桐油 1 週 / 層、3 週才完全硬化）、純品難取得、貴
- 市售「Tung Oil」很多其實是**油 + 樹脂混合**（不是純桐油）：
  - Minwax Tung Oil Finish = 含 PU 的油性 wiping varnish
  - Hope's Pure Tung Oil = 真的純桐油
  - Formby's Tung Oil = 含蟲膠的稀釋油

#### 桐油 SOP（純桐油）
1. 用礦物油精（mineral spirits）稀釋桐油 1:1（第一層）。
2. 擦遍工件、等 15-30 分鐘。
3. 擦掉多餘油。
4. **等 24-48 小時乾**。
5. 320 號輕磨。
6. 上第二層（不稀釋了）、重複擦掉、等乾。
7. 共 3-5 層、最後一層後等 1-2 週完全硬化才能用。

### 4.3 丹麥油（Danish Oil）

- **木頭仁新手家具入門推薦**
- 成分：50% 乾燥油（亞麻或桐）+ 30% 樹脂（醇酸或 PU）+ 20% 溶劑
- 優點：易上手、乾得快（24 小時 / 層）、滲入 + 略防水、不需要拋光
- 缺點：防護仍弱於 PU、需定期保養
- 代表品牌：Watco Danish Oil、Tried & True Danish Oil（食安級、純度更高）、Minwax Antique Oil
- SOP：抹 → 等 15 分鐘 → 擦掉 → 乾 24h → 砂 320 → 二度油 → 三度油

### 4.4 Tru-Oil（油性瓦尼斯）

- 原本是槍托用、後來吉他、刀柄、小件家具愛用
- 主要成分桐油 + 醇酸樹脂 + 礦物油
- 多薄層慢慢疊（每層 24h）、可疊到鏡面光澤
- 適合：小件、樂器、需要光澤的工件
- 不適合：大件家具（成本高、太亮）

### 4.5 礦物油（Mineral Oil）

- **食品安全等級**（FDA 認證）的白礦物油（white mineral oil）
- 用途：砧板、木碗、木匙、木盤
- **不會乾**（永遠是濕的），所以必須定期補（一個月 1-2 次）
- 不防水、不防熱、不耐刮 — 但對於「廚具用木」來說剛好
- 配合蜜蠟使用最佳（mineral oil + beeswax = butcher block conditioner）
- 注意：**買藥房 / 藥局的食品級 mineral oil**，不要買五金行的工業 mineral oil（含雜質、不可食）

### 4.6 硬蠟油（Hardwax Oil）— Osmo Polyx 系列

- 北歐 / 德國風、現在台灣木工最流行
- 成分：天然植物油（葵花、紅花、亞麻、薊花、大豆油）+ 巴西棕櫚蠟 + 蜜蠟
- 優點：滲入 + 表面成膜、防水中等、好保養、可重塗（不需要砂）
- 缺點：貴（一罐 750ml 約台幣 1500-2000）、需薄塗（厚塗會花）

#### Osmo Polyx-Oil 標準 SOP
1. 砂磨到 150-220（**不要超過 220**！砂太細 Osmo 滲不進去）。
2. 清潔。
3. 用 lint-free 棉布、海綿、或硬蠟油專用刷，**沾極少量**、**用力擦薄薄一層**。
4. 圓周運動 + 8 字運動把油帶開、確保無厚積。
5. 等 8-10 小時乾透（不能急）。
6. 用 320 號或灰色 scotchbrite 輕磨（去掉表面屑、增加附著）。
7. 上第二層，**更薄**。
8. 等 8-10 小時。
9. 一般家具 2 層夠用。地板可上到 3-4 層（用拋光機）。
10. 完全硬化需 **2-3 週**（這期間不要用濕布擦、不要放重物）。

> **Osmo 用太多最常見錯誤**：第二層擦太厚、油浮在表面、變黏 → 拿酒精 + 抹布擦掉重來，不要等乾。

### 4.7 油性塗料總比較表

| 產品 | 防水 | 防熱 | 食安 | 難度 | 變黃 | 乾燥 |
|------|------|------|------|------|------|------|
| BLO 亞麻仁油 | 差 | 差 | 否（含金屬乾燥劑） | 易 | 是 | 慢 |
| 純桐油 | 中 | 中 | 是 | 中 | 否 | 極慢 |
| 丹麥油 | 中 | 中 | 否（一般版） | 易 | 微 | 中 |
| Tried & True | 中 | 中 | 是（認證） | 中 | 否 | 慢 |
| Tru-Oil | 中 | 中 | 否 | 中 | 微 | 慢 |
| 礦物油 | 無 | 無 | 是 | 極易 | 否 | 不乾 |
| Osmo Polyx | 中-高 | 中 | 是（乾後） | 中 | 否 | 慢 |

## 第 5 章 蠟（Wax）

### 5.1 蠟的分類

| 蠟種 | 硬度 | 來源 | 用途 |
|------|------|------|------|
| 蜜蠟（beeswax） | 軟 | 蜜蜂 | 食品安全、保養油塗料、混合用 |
| 巴西棕櫚蠟（carnauba） | 極硬 | 巴西棕櫚樹葉 | 高光澤、車蠟、地板 |
| 調合蠟 / paste wax | 中 | 蜜蠟 + carnauba + 溶劑 | 家具拋光、現成最方便 |
| 微晶蠟（microcrystalline） | 中-硬 | 石油副產品 | 博物館保存、不變色 |

### 5.2 蠟的角色 — **不要單獨用蠟當主塗料**

蠟**不是塗料**、是**最後一層保護**。為什麼？
- 防水弱（碰水會變白霧）
- 不耐熱（手汗也會弱）
- 不耐磨（一個月就磨光）

**蠟的正確角色：**
- 上在油 / 蟲膠 / PU **之後**、增加光澤、抗指紋
- 古董修復、保留原皮殼
- 拋光木工車件（小件、不用怕磨損）
- 抽屜滑軌、燕尾榫 — 抹一點蠟讓它順

### 5.3 上蠟 SOP

1. 確認底層塗料**完全乾透**（油性 24h、PU 1-2 週）。
2. 用棉布或 0000 鋼絲絨**沾極少量**蠟，順木紋抹薄薄一層。
3. 等 10-15 分鐘（蠟裡的溶劑揮發）。
4. 乾淨棉布**用力拋光**到出現光澤。
5. 24 小時後可第二層（一般 1 層夠）。

### 5.4 自製蠟配方（木頭仁工作室可參考）

**經典 paste wax**（Bob Flexner 配方）：
- 蜜蠟 1 份（重量）
- carnauba 蠟 0.5 份
- 松節油 / 礦物油精 4 份（按需要調稀稠）
- 隔水加熱融化、攪拌、倒進鐵罐冷卻

## 第 6 章 蟲膠 / 紫膠（Shellac）

### 6.1 來源

蟲膠是**紫膠介殼蟲（Kerria lacca）**分泌物，主要產於印度、泰國。蟲膠片（flake）是去雜質後的純蟲膠，溶於酒精（denatured alcohol / 變性酒精 / 95% 乙醇）使用。

### 6.2 切片濃度（Cut）

「Cut」= 多少**磅**蟲膠片溶於 1 **加侖**酒精。

| Cut | 濃度 | 用途 |
|-----|------|------|
| 1lb cut | 淡 | French polish 起步、sealer 封閉劑 |
| 2lb cut | 中 | 一般刷塗、噴塗、家具標準 |
| 3lb cut | 厚 | 厚塗、補洞、家具表面層 |

**換算成台灣家用單位**：1 lb cut ≈ 28g 蟲膠片 / 200ml 酒精 ≈ 4 茶匙片 / 1 杯酒精。

### 6.3 蟲膠的特殊優勢

- **乾得快**：15-30 分鐘乾、可立刻上下一層
- **可疊塗**：每層會「咬合」前一層（酒精溶解前一層、化學鍵結）
- **可逆**：不滿意？酒精擦掉重來，**這是唯一能完全擦掉重來的塗料**
- **超佳 sealer**：蟲膠對任何塗料都有附著力（PU、油、水性、wax），可當底漆
- **無毒**：FDA 食品安全（藥丸糖衣就是蟲膠）
- **古董友善**：傳統法式拋光（French Polish）就是蟲膠

### 6.4 蟲膠的弱點

- **不耐熱**：> 65°C 會軟化（杯子放熱湯會留白印）
- **不耐酒精**：紅酒、香水、化妝水會溶解（酒杯墊一定要墊）
- **不耐鹼**：肥皂水擦多了會霧
- **保存期限短**：泡好的液體蟲膠**6 個月**後會失效（不再硬化），標日期！

### 6.5 蟲膠用途

- 古典家具、樂器（吉他、小提琴）
- French Polish（傳統手工拋光、需要功夫）
- Sealer 底漆（封節疤、防樹脂滲出）
- 修補小傷（蟲膠棒、棉花棒點塗）

### 6.6 標準刷塗 SOP（2lb cut）

1. 砂到 220。
2. 平刷（鬃毛刷）沾蟲膠液、**順木紋一刷到底**（不要來回）。
3. 等 30 分鐘乾。
4. 320 號輕磨。
5. 第二層、第三層 ... 一般 3-5 層。
6. 最後一層用 0000 鋼絲絨輕磨 + paste wax 拋光。

## 第 7 章 聚氨酯（Polyurethane, PU）

PU 是**目前家具最常用塗料**、防護最強、市面最多選擇。

### 7.1 油性 PU（Oil-based PU）

- 代表：Minwax Polyurethane、Varathane、Behlen、台灣虹牌油性 PU
- 防護：**最強**（耐水、耐熱、耐刮、耐酒精）
- 缺點：
  - **變黃**（隨時間越來越黃，特別是淺色木）
  - 味道重（VOC 高，需通風）
  - 乾得慢（12-24 小時 / 層）
  - 工具洗 mineral spirits

### 7.2 水性 PU（Waterborne PU）

- 代表：Minwax Polycrylic、General Finishes High Performance、台灣立邦水性、虹牌水性 PU
- 優點：
  - **不變黃**（白色 / 淺色家具必選）
  - 味淡、低 VOC
  - 乾得快（2-4 小時 / 層）
  - 清水洗工具
- 缺點：
  - 防護**略弱於油性**（但近年 GF High Performance 已經追上）
  - **會升毛**（必須先 raise grain）
  - 比油性貴

### 7.3 油性 vs 水性 PU 比較

| 項目 | 油性 PU | 水性 PU |
|------|---------|---------|
| 顏色 | 微黃、增添暖色 | 透明、不變色 |
| 變黃 | 是、年復一年 | 否（少數便宜貨會微黃） |
| 防水 | 強 | 中-強 |
| 防熱 | 強 | 中-強 |
| 防刮 | 強 | 中-強（GF High Perf 強） |
| 防酒精 / 化學 | 強 | 中 |
| 乾燥 | 12-24h / 層 | 2-6h / 層 |
| 完全硬化 | 30 天 | 21 天 |
| 味道 | 重 | 淡 |
| 工具清潔 | mineral spirits | 清水 |
| 成本 | 中 | 中-高 |
| 升毛 | 否 | 是 |

**白話建議**：
- 餐桌、廚房櫃台、地板、深色家具 → 油性 PU
- 白色家具、淺色木（楓、白橡）、嬰兒家具、室內細木工 → 水性 PU
- 想要「拋光鏡面」 → 噴塗版本、層層砂磨

### 7.4 光澤選擇

| 光澤 | 英文 | 反光 | 用途 |
|------|------|------|------|
| 高光 / 亮光 | Gloss | 90+ | 樂器、鋼琴、藝術品 |
| 半亮 | Semi-gloss | 50-70 | 廚櫃、門 |
| 緞光 / 半光 | Satin | 30-40 | 一般家具（最受歡迎） |
| 蛋殼光 | Eggshell | 20-30 | 牆面、低調家具 |
| 啞光 / 平光 | Matte / Flat | < 10 | 北歐風、隱藏木紋瑕疵 |

### 7.5 標準 PU 上漆 SOP

1. 砂到 220。清潔（壓縮空氣 + 黏塵布）。
2. 第一層稀釋 50%（油性用 mineral spirits、水性用清水），當 sealer 用。
3. 等 4-12 小時乾（看油 / 水）。
4. 320 號**輕磨**（去掉浮塵 + 升毛）、清潔。
5. 第二層**不稀釋**、薄塗。
6. 等乾、320 號磨、清潔。
7. 第三層 = 最後層、**最薄**。
8. **不要砂最後一層**！直接讓它自然乾、自然光澤。
9. 完全硬化（油性 30 天、水性 21 天）才能上重物 / 用濕布擦。

### 7.6 PU 常見錯誤

- 上太厚 → 流掛、橘皮
- 不砂中間層 → 第二層不黏、剝落
- 沒等乾 → 起泡、白霧
- 低溫塗（< 10°C）→ 不固化
- 工件沾灰 → 沙粒卡進塗料、永遠拿不掉

## 第 8 章 噴塗 vs 刷塗 vs 擦塗

### 8.1 噴塗（Spray）

#### 噴槍類型

| 類型 | 壓力 | 用量 | 適用 |
|------|------|------|------|
| HVLP（High Volume Low Pressure） | 低 | 高效 | **家用最佳**、低過噴 |
| Conventional / 高壓 | 高 | 低效 | 工業、車漆 |
| Airless / 無氣 | 極高 | 高 | 大面積牆漆、戶外 |
| LVLP | 低 | 中 | 小工作室、低 CFM 壓縮機 |

#### HVLP 噴嘴選擇

| 噴嘴 | 適用塗料 |
|------|----------|
| 1.0-1.2mm | 染料、超薄塗料、樂器 |
| 1.3mm | **一般 PU、清漆、水性塗料**（最常用） |
| 1.4-1.5mm | 中黏度油性、有顏色塗料 |
| 1.8mm | 厚漆、底漆、白漆 |
| 2.0+mm | 乳膠漆、大面積 |

#### 噴塗技術重點

- 距離工件 **15-20cm**（6-8 寸）
- 重疊 **50%** 移動
- 速度均勻、不停留
- 先試噴在紙板上（試橘皮、試流掛）
- 噴室通風 + 防塵（兩難 → 自製噴漆帳 / 雨後清晨噴）

### 8.2 刷塗（Brush）

#### 刷子選擇

| 塗料 | 刷種 |
|------|------|
| 油性 PU、油性染色、油性塗料 | **天然鬃毛**（豬鬃、馬鬃） |
| 水性 PU、水性染色 | **合成毛**（尼龍 / 聚酯混合） |
| 蟲膠 | 鬃毛 / 駱駝毛軟刷 |

**禁忌**：水性塗料用天然鬃毛 → 鬃毛吸水變鬆、刷痕嚴重。

#### 刷塗技術

1. 沾刷時**只浸前 1/3**、輕敲刷柄不要刮邊。
2. 順木紋一刷到底、不要來回。
3. 一個方向重疊上一刷的「濕邊」、避免結痕。
4. 最後一刷叫「**tipping off**」：空刷（無漆）輕掃表面消除刷痕。
5. 大面積分區處理、保持 wet edge。

### 8.3 擦塗（Wipe-on）

把油性 PU 用礦物油精稀釋 50%、用棉布擦上去。
- 優點：**不留刷痕**、新手友善、工具便宜（一條布）
- 缺點：層數要多（每層比刷塗薄一半）、慢
- 適合：小件、雕件、複雜形狀
- 代表：Minwax Wipe-On Poly、Watco Wipe-On

## 第 9 章 食品安全塗料

### 9.1 真正食品安全的塗料（FDA / 認證）

| 塗料 | 適用 | 備註 |
|------|------|------|
| 食品級礦物油（white mineral oil） | 砧板、木碗、木匙 | 藥房購買、永遠不乾 |
| 蜜蠟（純） | 砧板、玩具 | 配合礦物油使用最佳 |
| Howard Butcher Block Conditioner | 砧板 | 礦物油 + 蜜蠟 + carnauba |
| Walrus Oil（NSF 認證） | 砧板、餐具 | 美國食安認證 |
| Tried & True Original / Danish Oil | 玩具、餐具 | 純亞麻仁油聚合品、無金屬乾燥劑 |
| 純胡桃油 / 純椰子油 | 砧板（不過敏者） | 會慢慢乾 |
| Osmo Top Oil（食品級系列） | 廚房檯面 | 乾後安全 |
| 純蜜蠟 / 純 carnauba | 玩具 | 嬰兒玩具最常用 |
| 純桐油（Hope's Pure Tung Oil） | 餐具 | 乾燥慢但無毒 |

### 9.2 一般 PU 塗料完全乾後也算食品安全？

**理論上是。** FDA 規定：所有塗料**完全硬化後**（cured，不只是 dry）都是食品安全的。所以 Minwax PU、General Finishes 等等家用塗料，等 **30 天完全硬化**後，理論上可以用來做砧板。

**但實務上不建議**，因為：
- 砧板會被**刀切**、塗層碎屑會混進食物
- 砧板會**碰熱湯**、PU 不耐熱
- 砧板會**洗碗精**、化學物會慢慢分解塗層

**結論**：
- 砧板 / 切菜板 → 礦物油 + 蜜蠟（每月補）
- 木碗、木匙 → Tried & True 或純桐油
- 木製餐盤（不切割）→ 一般 PU 完全乾後可以、但寧可用 Osmo Top Oil

### 9.3 嬰兒玩具

最高安全等級：
- Tried & True Original（無金屬乾燥劑）
- 純蜜蠟 / 純 carnauba
- Osmo Top Oil 食品級
- EN71-3 認證塗料（歐盟玩具安全標準）

**不要用**：油性 PU（含催化劑）、有機溶劑染料、含鉛顏料（古董漆）。

## 第 10 章 戶外塗料

### 10.1 戶外塗料的關鍵：UV 抗性 + 彈性 + 防水

戶外塗料**不能用室內 PU**！原因：
- 室內 PU 沒 UV 抑制劑 → 太陽下半年就裂
- 室內 PU 太硬 → 木材熱脹冷縮會裂塗層

### 10.2 主要戶外塗料

| 塗料 | 防 UV | 彈性 | 用途 |
|------|-------|------|------|
| Spar Varnish / Spar Urethane | 強 | 高 | 戶外家具、船甲板、門 |
| Sikkens Cetol | 強 | 高 | 戶外家具、windows |
| Penofin（penetrating oil） | 中-強 | 高 | 戶外硬木、deck |
| Cabot 戶外油 | 中 | 高 | deck、圍籬 |
| Behr 戶外染色 + 封閉劑 | 中 | 中 | 一般家用 deck |
| 柚木油（Teak Oil） | 中 | 高 | 柚木、油性硬木 |
| Marine Varnish（兩液型） | 極強 | 高 | 船、長期戶外 |

### 10.3 柚木油（Teak Oil）

柚木**自身含油**（natural oils），不需要「保護」、但塗 teak oil 可以：
- 突顯紋路、保留金黃色
- 略防 UV（防止灰化）
- 不上會慢慢變灰銀色（很多人故意要這效果）

不要把 teak oil 用在非柚木上 — 它的配方對柚木最佳。

### 10.4 戶外塗裝禁忌

- **不可用**：純亞麻仁油（不防 UV、易發霉）
- **不可用**：室內 PU（裂、剝落）
- **不可用**：蟲膠（不防水、不防 UV）
- **不可用**：純桐油（防 UV 弱）
- **不可用**：Osmo 室內版（要用 Osmo UV-Schutz Öl 或 Country Color）

### 10.5 戶外保養週期

- Spar Varnish / Marine：每 2-3 年砂磨重塗
- 戶外油（Penofin / Cabot）：每 1-2 年補塗（不需砂）
- Teak Oil：每 6-12 個月補
- Deck 染色：每 2-3 年重塗

## 第 11 章 塗裝缺陷與排除

### 11.1 流掛（Run / Sag / Drip）

- **症狀**：塗料像眼淚一樣垂下來、形成淚珠
- **原因**：塗太厚、垂直面、稀釋過度、刷太用力
- **對策**：等完全乾 → 砂掉（從 220 開始）→ 重新薄塗
- **預防**：垂直面分次薄塗、刷 / 噴薄一點、最後一刷 tipping off

### 11.2 橘皮（Orange Peel）

- **症狀**：表面像橘子皮、有小坑
- **原因**：噴塗距離太近、漆太厚、空氣壓力不對、漆太黏
- **對策**：完全乾 → 320 砂平 → 400 → 重噴薄一層
- **預防**：噴前試噴紙板、稀釋到正確黏度、距離 15-20cm

### 11.3 魚眼（Fish Eye）

- **症狀**：表面有像月球坑的圓形凹點、塗料分開
- **原因**：表面有**矽油 / 蠟 / 油**污染（矽是頭號嫌犯）
- **常見來源**：噴霧家具蠟（Pledge）、WD-40、矽脫模劑、修補蠟、有些家具拋光劑
- **對策**：
  1. 砂掉到木材 → 用變性酒精 + 棉布徹底擦 → 蟲膠 1lb cut 兩層當 barrier coat → 重塗
  2. 加 fish-eye eliminator（含矽流平劑）到塗料中 → 反向利用矽降低表面張力
- **預防**：噴塗前**永遠**用酒精擦表面、不要在用過矽油的工作台旁噴漆

### 11.4 起泡（Blistering / Bubbling）

- **症狀**：塗層下有氣泡、按壓會破
- **原因**：底材含水太高、太陽下塗（溶劑揮發太快但表層先封）、底層未乾就塗下一層
- **對策**：刺破 → 砂平 → 重塗
- **預防**：木材含水率 < 12% 才塗、避開大太陽 / 高溫塗、層間等乾透

### 11.5 變色 / 黃變

- **油性 PU 變黃**：自然現象、無解，要不變黃就用水性
- **白漆變黃**：油性 PU 永遠會把白漆變奶油色 → 白家具用水性
- **氧化變色**：櫻桃、胡桃見光會深、保留無紫外線環境減緩

### 11.6 白霧 / 水印（Water Spot / Blushing）

- **症狀**：透明塗層下有白色霧斑
- **原因**：高濕度下塗（水分被困）、熱杯子放上去（不耐熱）
- **對策**：白霧 → 白蘭地 / 變性酒精擦（蟲膠用）、吹風機加熱（PU 用）、嚴重要砂掉重塗
- **預防**：濕度 < 65% 塗、杯墊永遠用

### 11.7 刷痕

- **原因**：刷太慢、來回刷、塗料太黏、刷子毛太粗
- **對策**：tipping off、稀釋塗料、換軟刷（pure white China bristle）、改擦塗

### 11.8 灰塵 nibs（卡進塗層的小顆粒）

- **症狀**：塗層裡卡了灰塵、看起來像痘痘
- **對策**：等乾 → 320 / 400 輕磨平 → 上下一層
- **預防**：噴前清潔工件、噴漆室加濕（壓灰）、衣服脫毛、停冷氣

## 第 12 章 塗裝環境

### 12.1 理想條件

| 項目 | 範圍 |
|------|------|
| 溫度 | 15-25°C |
| 濕度 | 40-65% |
| 通風 | 良好（除溶劑味）但無強風（揚塵） |
| 光線 | 充足（看出橘皮、流掛） |
| 灰塵 | 最少（噴塗最關鍵） |

台灣氣候特殊：
- **夏天梅雨**（濕度 > 80%）：水性塗料乾很慢、易白霧。要用除濕機 + 冷氣壓濕度。
- **冬天寒流**（< 10°C）：油性 PU 不固化。要等回溫 / 加熱器。
- **春秋**：最佳塗裝季。

### 12.2 通風 vs 防塵的矛盾

塗裝的兩難：
- 要通風（除溶劑、健康、易乾）
- 要無塵（避免灰塵卡進塗層）

**解法：**
1. **噴塗後立刻關窗**：噴的時候開、噴完關、讓塗料在無塵環境乾
2. **自製噴漆帳**：用 PVC 管 + 塑膠布、貼紙箱地板
3. **塑膠布拉門 / 浴簾**隔開噴塗區
4. **溼布拖地**：噴前 30 分鐘溼拖、灰塵被壓住
5. **戴口罩 + 護目鏡**（噴 PU、清漆、硝化漆都需要 organic vapor 等級口罩）

### 12.3 工具清潔

| 塗料 | 清潔劑 |
|------|--------|
| 油性 PU、油性染色、Tru-Oil、Watco | mineral spirits、礦物油精 |
| 水性 PU、水性染色、Aqua Coat | 清水（立刻洗、乾了難洗） |
| 蟲膠 | 變性酒精 / 95% 乙醇 |
| 環氧樹脂 / AB 膠 | 醋酸乙酯（洗甲水）、酒精（未硬化） |

## 第 12.5 章 塗料與化學廢料處理（台灣法規 + 實務 SOP）

### 12.5.1 各類廢料處理對照

| 廢料 | 不可丟一般垃圾/水溝 | 正確處置 |
|---|---|---|
| **油性 PU 剩漆** | ❌ | 攤開風乾固化 → 鐵罐密封 → **環保局有害廢棄物收集日** / 1999 詢問 |
| **水性 PU 剩漆** | ❌（環境影響較小但仍含化學） | 加吸附材（貓砂/木屑）固化 → 一般垃圾 / 環保局收 |
| **稀釋劑（mineral spirits、香蕉水、丙酮）** | ❌（極危險） | **絕不可倒水溝**！油性溶劑漂浮在水面易引燃 → 鐵罐密封 → 環保局有害廢日 |
| **油布（亞麻仁油、桐油、丹麥油）** | ⚠️ 自燃風險 | **泡水裝鐵罐密封** 或 **單張攤開於不可燃面風乾完全氧化** → 一般垃圾 |
| **染色劑（pigment / dye 廢液）** | ❌ | 吸附材固化 → 環保局有害廢日 |
| **環氧樹脂（未硬化）** | ❌ | A、B 劑混合讓它完全硬化（會放熱、要在通風處） → 硬化後可一般垃圾 |
| **空塗料鐵罐** | 漆殼乾透後可資源回收 | 蓋打開風乾 → 鐵罐回收 |
| **空溶劑塑膠瓶** | ❌（含殘液） | 同類溶劑沖洗瓶身、廢液收集 → 瓶身塑膠回收 |

### 12.5.2 台灣處理管道

- **環保局 1999**：詢問所在縣市的「家戶有害廢棄物收集日」（多為每月或每季一次，免費）
- **資源回收車**：油漆鐵罐（乾透後）可回收金屬類
- **業界處理商**：大量者找「事業廢棄物清除處理機構」（合法業者列表在環保署網站）
- **絕對不可**：倒馬桶、倒水溝、混入一般垃圾、倒河川/排水溝、燒掉（產生戴奧辛）

### 12.5.3 累積與儲存

- 油性溶劑、油性 PU 廢料：**金屬桶 + 緊密蓋**，遠離火源、陽光直射
- 累積到一桶（5–10L）一次送收集日，平時別亂丟
- 小量擦拭過油的紗布/抹布：**先泡水再丟**，或攤開在水泥地完全氧化（不堆疊）

### 12.5.4 法規參考

- 《廢棄物清理法》：家戶有害廢棄物應分類收集
- 《毒性及關注化學物質管理法》：含揮發性有機物（VOC）廢料需申報
- 業者：環保署 EMS 申報系統（年產 100kg 以上）

## 第 15 章 木頭仁實戰塗裝決策樹

```
什麼木？  → 開孔木（橡、白蠟）→ 考慮 grain filler
            → 軟木（松、檜）→ 用 pre-stain conditioner
            → 易花染（楓、櫻桃）→ gel stain 或 dye + conditioner

什麼用途？→ 砧板 / 餐具 → 礦物油 + 蜜蠟（最簡單）/ Tried & True
            → 餐桌（會放熱湯酒）→ 油性 PU 多層（防護優先）
            → 客廳家具 → Osmo Polyx 或 水性 PU（外觀優先）
            → 白色家具 → 水性 PU（不變黃）
            → 戶外 → Spar Varnish / Penofin
            → 古典 / 樂器 → 蟲膠 + 蠟
            → 嬰兒玩具 → Tried & True 或純蜜蠟

什麼風格？→ 北歐極簡 → Osmo / 硬蠟油（霧面、自然）
            → 工業風 / 復古 → BLO + 蠟、鐵醋液染色
            → 鏡面 / 古典 → 蟲膠 + 法式拋光、或多層 PU + 拋光
            → 原木感 → 純桐油、Osmo Top Oil

預算？    → 高 → Osmo / Tried & True / GF High Performance
            → 中 → Watco Danish Oil / 虹牌油性 PU
            → 低 → 立邦水性 PU / Minwax 入門 / 蟲膠片自配
```
